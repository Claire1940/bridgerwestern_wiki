<Section icon={Map} title="Como Obter a Carta Evil Eye">

Garantir o **bridger western evil eye** exige uma viagem a uma das zonas mais perigosas do mapa: a Cidade Abandonada (Abandoned Town). Este local não é apenas um cenário para tiroteios; ele abriga os NPCs essenciais necessários para a progressão no meio do jogo.

<Steps>
  <Step num="1" title="Chegue à Cidade Abandonada">
    Viaje para o setor norte do mapa. Procure pelo aglomerado de edifícios em ruínas. É altamente recomendável usar um cavalo para esta jornada para evitar ser pego por Vampiros errantes ou caçadores de PvP.
  </Step>
  <Step num="2" title="Localize o NPC Bridgerman">
    Atrás de uma das estruturas principais, você encontrará um NPC chamado **Bridgerman**. Ele fica perto da área dedicada ao giro de cartas. Ele é a única fonte para comprar novas passivas de cartas.
  </Step>
  <Step num="3" title="Prepare sua Moola">
    Cada giro custa exatamente **450 Moola**. Como o conjunto de cartas inclui quase 20 opções diferentes, você deve trazer pelo menos 2.000 a 3.000 Moola para aumentar suas chances de conseguir o Evil Eye.
  </Step>
  <Step num="4" title="Verifique seu Tier">
    Abra o menu do seu personagem para verificar seu Tier atual. Se você for Tier 1, poderá equipar apenas o Evil Eye. Se for Tier 3, poderá combiná-lo com outras duas cartas.
  </Step>
</Steps>

<Callout type="warning" title="Restrição para Vampiros">
Se você já se comprometeu com o caminho de progressão de Vampiro, o **bridger western evil eye** será inútil para você. As mecânicas do jogo afirmam explicitamente que Vampiros não podem utilizar o sistema de cartas.
</Callout>

</Section>
