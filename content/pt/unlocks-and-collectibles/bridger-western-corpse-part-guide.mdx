## Entendendo as Mecânicas das Partes do Cadáver

Antes de se aventurar na natureza, você deve entender como funciona o sistema de spawn. Ao contrário do saque padrão ou das recompensas de pesca, as Partes do Cadáver são eventos mundiais que ocorrem globalmente em todo o servidor.

Quando uma Parte do Cadáver aparece, um texto de notificação surgirá no céu, seguido por um enorme feixe de luz que é visível de quase qualquer lugar do mapa. Este feixe serve como um farol para todos os jogadores na sessão. Assim que um jogador pega a parte, o feixe desaparece, mas a luta do portador está apenas começando.

| Mecânica | Descrição |
| :--- | :--- |
| **Notificação Global** | Um texto aparece no céu sinalizando que um spawn ocorreu. |
| **Sinalizador Visual** | Um feixe de luz brilhante marca a localização exata da parte. |
| **O Brilho (The Shine)** | Os portadores pulsam com uma luz branca a cada poucos segundos, revelando sua posição. |
| **Soltar ao Morrer** | Se um jogador que segura uma parte for morto, o item cai no chão para outros. |
| **Fusão** | A cada poucos segundos, há uma chance de a parte se "fundir" com o portador, concedendo um Stand. |

> **⚠️ Aviso:** Pegar uma Parte do Cadáver torna você o alvo principal. O efeito "Brilho" garante que você não consiga se esconder por muito tempo, portanto, ter um cavalo rápido é obrigatório para a sobrevivência.

## Todos os Locais de Spawn das Partes do Cadáver

Encontrar uma parte requer conhecer os pontos quentes do mapa. Embora os spawns possam parecer aleatórios, eles ocorrem em pontos fixos específicos. Usar esta lista do **guia de partes do cadáver de bridger western** permitirá que você se posicione perto de vários locais potenciais para minimizar o tempo de viagem.

### 1. Taco Bay e Vila de Pescadores (Fishing Village)
Esta é indiscutivelmente a área mais ativa para spawns. Existem três pontos distintos aqui:
*   **A Torre:** Uma parte pode aparecer no topo da torre de madeira em Taco Bay.
*   **A Frente da Loja:** Diretamente em frente à loja de Taco Bay.
*   **A Área da Ponte:** Perto da ponte que conecta a vila ao continente.

### 2. O Local de Escavação (The Dig Site)
Localizado perto do Condado Rich B, o Dig Site é um local de alto risco. A parte aparece no fundo do poço de escavação. 
*   **Fator de Risco:** Alto. Você deve usar uma corda para sair, tornando-se um alvo fácil para jogadores que esperam no topo do poço.

### 3. Condado Rich B (Cumes Superiores)
Perto das fronteiras do condado, as partes costumam aparecer nos planaltos rochosos elevados. Estes são mais seguros que o Dig Site porque oferecem múltiplas rotas de fuga pelos cumes.

### 4. O Deserto (Rocha Escondida)
Nas planícies abertas entre as cidades principais, há um ponto de spawn escondido atrás de uma grande formação rochosa. Este é um achado baseado na "sorte" para jogadores que por acaso estão viajando entre as zonas.

### 5. O Vale (The Valley)
Este é o ponto de spawn mais distante dos centros principais. Embora seja menos lotado, a distância dificulta o alcance se você ainda não estiver por perto. Pegar o trem é a maneira mais rápida de chegar a esta zona.

| Localização | Dificuldade | Melhor Rota de Fuga |
| :--- | :--- | :--- |
| **Torre de Taco Bay** | Média | Pule e siga em direção aos trilhos do trem. |
| **Poço do Dig Site** | Extrema | Requer uma subida rápida; altamente suscetível a campistas. |
| **Alturas de Rich B** | Baixa | Cavalgue pelo deserto aberto em direção ao Hotel. |
| **O Vale** | Alta | Use o trem ou siga o leito do rio. |

## Como Obter D4C (Dirty Deeds Done Dirt Cheap)

Embora a maioria dos Stands possa ser obtida através do processo de fusão padrão ou Stand Arrows (fragmentos da pesca), o **D4C** requer um ritual muito mais específico e exaustivo. Simplesmente encontrar uma parte não é suficiente.

Para desbloquear o D4C, você deve seguir estes passos exatos:
1.  **Acumular 100 Abates Totais:** Esta é uma estatística vitalícia em todos os servidores.
2.  **Alcançar um Killstreak de 10 Jogadores:** Isso deve ser feito em um único servidor sem morrer.
3.  **Ir para o Deserto:** Assim que tiver a sequência, vá para o bioma do deserto.
4.  **Ser Derrubado (Knocked):** Você deve estar no estado "derrubado" enquanto mantém seu killstreak.
5.  **Evento de Spawn:** Enquanto estiver derrubado, uma Parte do Cadáver deve aparecer no mundo.
6.  **A Fusão do Coração:** Há uma **chance de 10% de RNG** de que a parte do "Coração" se funda com você enquanto você estiver derrubado, concedendo o D4C.

> **💡 Dica:** Devido aos 10% de RNG, muitos jogadores falham nisso várias vezes. Recomenda-se ter um amigo para protegê-lo enquanto você estiver derrubado no deserto para evitar que outros jogadores o finalizem.

## Equipamento Recomendado para Caçar Partes

Se você pretende seguir este **guia de partes do cadáver de bridger western** com sucesso, não pode sair com roupas básicas. Você precisa de equipamentos que mitiguem o dano e aumentem sua mobilidade.

*   **O Chapéu de Cowboy:** Comprado na Loja Geral. Ele atua como um escudo de "um acerto" para tiros na cabeça. O primeiro tiro na sua cabeça simplesmente derrubará o chapéu, causando dano zero.
*   **O Poncho:** Este item reduz o dano recebido no tronco. Embora aumente ligeiramente o dano nos membros, a maioria dos jogadores mira no centro de massa, tornando o Poncho um ganho líquido para a sobrevivência.
*   **Rifle Winchester:** Alta precisão e dano são necessários para afastar perseguidores enquanto você espera por uma fusão de parte.
*   **Cavalo de Alta Velocidade:** Visite os estábulos e procure por um cavalo com uma descrição que enfatize a velocidade. Um cavalo que custa mais de 1.000 Moola é geralmente a base para o jogo competitivo.

| Item | Custo | Benefício Principal |
| :--- | :--- | :--- |
| **Chapéu de Cowboy** | Barato | Bloqueia inteiramente um tiro na cabeça. |
| **Poncho** | Moderado | Reduz significativamente o dano no tronco. |
| **Whitney Dragon** | Alto | Excelente arma secundária para defesa em combate próximo. |
| **Cavalo Veloz** | 1.000+ | Essencial para fugir dos caçadores atraídos pelo "Brilho". |

## FAQ

**P: Quanto tempo leva para uma Parte do Cadáver se fundir?**

R: Não há um tempo fixo. A cada poucos segundos que você segura a parte, o jogo rola uma chance de fusão. Pode acontecer em 10 segundos ou levar vários minutos sobrevivendo ao "Brilho".

**P: Posso perder meu Stand se eu morrer?**

R: Não, uma vez que um Stand é fundido ao seu personagem, ele é permanente. Você só pode trocá-lo ou removê-lo visitando o NPC Bruxa (Witch) e pagando Moola para resetar seu Stand.

**P: Qual é a melhor maneira de encontrar uma parte se eu for um jogador solo?**

R: Use este **guia de partes do cadáver de bridger western** para identificar o spawn da Torre de Taco Bay. É o mais fácil de avistar à distância. Fique perto dos trilhos do trem para que possa se mover rapidamente entre a Vila de Pescadores e os spawns do Deserto.

**P: A habilidade "Ocular Prowess" ajuda a encontrar Partes do Cadáver?**

R: O Ocular Prowess ajuda você a ver destaques de jogadores, o que é útil para localizar a pessoa que *tem* a parte, mas não destaca a parte em si antes de ser coletada. Use o feixe de luz para a caça inicial.
