## All Spin Locations Bridger Western のマスター：スピンロック

旅の最初のステップは、スピンのエネルギーに「選ばれる」ことです。このエネルギーは、マップ上に点在する特定の岩の形成物に現れます。これらの岩は常にエネルギーを持っているわけではなく、ランダムに青い渦巻くオーラで光り出します。このステップを完了するには、アクティブな岩を見つけ、そのエネルギーの中に足を踏み入れる必要があります。

このステップのために **all spin locations bridger western** が提供する6つの特定のエリアを確認してください：

1.  **ザ・キャニオン（渓谷）:** 岩だらけの渓谷の回廊内にある高い岩棚の上にあります。
2.  **発掘現場（The Dig Site）:** 掘削機器の近くにある大きな平らな岩を探してください。
3.  **メインタウン:** 町に入り、右折して、人里離れた境界付近にあるスパイラルエネルギーを探します。
4.  **湖:** 水辺を見下ろす目立つ岩の上に位置しています。
5.  **沼地（魔女の小屋）:** 魔女の小屋の裏側に回り、暗い森の中を直進すると隠された岩が見つかります。
6.  **アウトローパス:** アウトローの隠れ家から左へ進みます。岩は高台の尾根にあります。

> **💡 ヒント:** 時間を節約するために、多くのプレイヤーは「サーバーホッピング（サーバーの切り替え）」を推奨しています。出現場所の一つに行き、エネルギーがなければ、アクティブな場所が見つかるまで別のサーバーに参加し直します。

## ジャイロ・ツェペリを見つける：出現場所

スピンのエネルギーに触れた後の次のタスクは、師匠であるジャイロ・ツェペリ自身を見つけることです。スピンロックと同様に、ジャイロにも6つの潜在的な出現場所があります。彼は象徴的な服装と、持っている鉄球で見分けることができます。

| 場所名 | 具体的な目印 |
| :--- | :--- |
| **キャニオン（渓谷）** | 渓谷の底深く（時々バグで増殖していることがあります） |
| **町の西** | メイン居住区の左側にある大きなサボテンの隣 |
| **タウンセンター** | 町の中心部にある建物の近くに時々出現します |
| **リンゴォの家** | 沼地に隣接する家の近くにある木の下で休んでいます |
| **沼地の教会** | 廃墟となった教会の中（魔女の小屋から直進） |
| **アウトローの尾根** | アウトローの隠れ家近くにある小さな家 |

ジャイロを見つけたら、彼に話しかけてください。彼は「拳（ナックル）を鍛えてこい」と言い、これによりレッドコーナーでのクエストの次のフェーズが始まります。

## レッドコーナー：20キルストリーク・チャレンジ

レッドコーナーは、トンネル内にある隠されたファイトクラブです。見つけるには、釣り船のドックから始まる線路を辿ってください。トンネル内にある小さく目立たないドアを探してください。

ファイトクラブ内では、**20キルストリーク** を達成しなければなりません。これは、一度も死なずに20人のプレイヤーを倒す必要があることを意味します。倒されると、カウンターはゼロにリセットされます。

*   **ファーミング戦略:** 多くのプレイヤーは、これを早めるためにサブアカウントや友人を利用します。友人がアウトローであれば、アウトローキャンプに出現地点を設定することで、ファイトクラブに早く戻ることができます。
*   **追跡:** 画面の右下を見て、現在のストリークを確認してください。
*   **安全:** 20キルに達したと確信するまで、ファイトクラブのエリアを離れないでください。

## 真の男の世界を生き抜く

「鍛えた拳」を持ってジャイロのもとに戻ると、彼は **デュエリストの証（Duelist's Mark）** を要求します。これはクエストの中で最も困難で、運（RNG）に左右される部分です。「真の男（True Man）」というNPCが自分の近くに出現するのを待つ必要があります。

### 真の男を出現させて倒す方法
真の男は、山賊やゾンビが通常現れるエリア（リンゴォの家など）に出現します。出現率は非常に低く、プレイヤーによっては5時間以上待ったという報告もあります。

1.  **通知:** 画面にメッセージが表示され、**すべてのツールを外す** ように指示されます。これを怠ると、彼は即座に敵対するか、消えてしまう可能性があります。
2.  **外見:** 彼は標準的なNPCのように見えますが、遠くから不気味にこちらを見つめています。出現してから約30秒で消えてしまいます。
3.  **戦闘:** 拳または銃で倒すことができます。ただし、スタンドや高ティアの武器を取り出すと、彼は驚異的な正確さとスピードで反撃してきます。
4.  **勝利:** 死亡時に、彼は「真の男の世界へようこそ（Welcome to a true man's world）」と言います。これでこのステップは完了です。

> **⚠️ 警告:** 他のプレイヤーのために出現した真の男を倒しても意味がありません。解放の進捗にカウントされるには、あなた自身の個人的な遭遇である必要があります。

## 最終要件：キル数と現金

最後の追い込みには、**合計100キル** と **$25,000** の2つが必要です。

レッドコーナーのステップとは異なり、100キルはストリークである必要はありません。これらはマップ全体での通常のプレイヤーキルです。ゲーム内に累計キル数のトラッカーがないため、手動で数えておくことをお勧めします。すでに20キルストリークを完了している場合は、あと80キル必要です。

キル数と現金が揃ったら、ジャイロのもとに戻ります。もし彼がまだ「戦いによって鍛えられる必要がある」と言う場合は、単にキル数が足りていないだけです。すべての条件が満たされると、スピンのスタイルを購入できます。能力がすぐに現れない場合は、**キャラクターをリセット** してインベントリを更新してください。

## FAQ

**Q: 最初のエネルギーを得るための Bridger Western の全スピン場所はどこですか？**

A: キャニオン、発掘現場、町（右側）、湖、沼地（魔女の小屋の裏）、アウトローの隠れ家近くの6か所です。光る青い渦巻き状のエネルギーを探してください。

**Q: なぜジャイロは話してくれないのですか？**

A: 主な理由は2つあります。1つ目は、ゲーム内年齢が40歳未満である必要があること。2つ目は、初めて彼に話しかける前に、光るスピンロックに触れている必要があることです。

**Q: 一度入手したスピンを外すことはできますか？**

A: 現在、一度購入したスピンを削除する方法はありません。鉄球の投射物のために銃を使う能力を捨てる覚悟があるか、決める前に慎重に検討してください。

**Q: 100キルの要件にファイトクラブでのキルは含まれますか？**

A: はい、ファイトクラブのストリーク中に達成した20キルも、合計100キルの要件にカウントされます。その後、さらにおよそ80キルが必要になります。
