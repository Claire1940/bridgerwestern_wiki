## 基本操作と最初のステップ

遺体パーツ（Corpse Parts）を探し始める前に、移動と戦闘の基本をマスターする必要があります。Bridger Westernは標準的な三人称視点の操作体系を採用していますが、探索とPvPの両方で不可欠な独自の要素がいくつか追加されています。

| コマンド | アクション | 説明 |
| :--- | :--- | :--- |
| **Wを2回押す** | ダッシュ | 町の間の移動に不可欠。障害物を自動的に乗り越えます。 |
| **Qキー** | ローリング | 短い無敵時間を提供し、位置取りを変えることができます。 |
| **M1 / 右クリック** | 射撃 / 照準 | すべての火器と弓の標準的な戦闘操作です。 |
| **Rキー** | リロード | 手動で武器をリロードします。カードで速度を強化可能です。 |
| **Xキー** | Ocular Prowess | 敵をハイライトし、命中精度とリロード速度を向上させます。 |
| **Hキー** | 馬を呼ぶ | 現在使用している馬を即座に自分の場所に呼び寄せます。 |
| **Gキー** | 近接攻撃 | 装備している武器で物理的な打撃を与えます。 |

スポーンしたら、最初の目的は線路を見つけることです。この線路はマップの主要なナビゲーションツールとして機能し、リッジビー郡やフィッシング・ヴィレッジ（Fishing Village）などの主要拠点へと続いています。線路に沿って進むのが、広大な砂漠や沼地のバイオームで迷うことなく方向感覚を掴むための、初心者にとって最も安全な方法です。

## 釣り経済：Moolaを素早く稼ぐ方法

どのような**Bridger Western初心者向けチュートリアル**においても、最も一貫したアドバイスは「フィッシング・ヴィレッジへ直行せよ」というものです。山賊やゾンビとの戦闘も実入りが良いですが、釣りは「Moola」として知られる初期資金を構築するための、最も安全で効率的な方法です。

1.  **Taco Bait Shopを探す：** フィッシング・ヴィレッジ近くの橋にたどり着くまで線路をたどります。
2.  **装備を購入する：** 小屋の中にいるNPCに話しかけ、釣り竿と餌を購入します。
3.  **ミニゲームを開始する：** 水辺に立ち、糸を垂らします。魚が食いついたら、制限時間内に特定のキーを押すスキルチェックを完了させる必要があります。
4.  **獲物を売る：** NPCのところに戻って魚を売ります。珍しい魚が釣れれば、1回のセッションで数千Moolaを稼ぐことも可能です。

> **💡 ヒント：** 釣りをしている最中、0.5%の確率でスタンドの矢（Stand Arrow）や破片（Shards）が釣れることがあります。これは、遺体パーツを巡る危険なPvPに身を投じることなく、最初のスタンドを獲得できる「平和的」な方法です。

## 必須装備とリッジビー郡のサービス

数千Moolaが貯まったら、リッジビー郡へ向かいましょう。この町は主要な交流と商業の拠点です。生存率を高めるために、すぐに防御装備を買うことを優先すべきです。

| アイテム | 場所 | メリット |
| :--- | :--- | :--- |
| **カウボーイハット** | 雑貨店 | 壊れる前に**ヘッドショット1回分**のダメージをゼロにします。 |
| **ポンチョ** | 雑貨店 | **胴体**へのダメージを軽減しますが、手足へのダメージが増加します。 |
| **トニック** | 医者/商店 | 戦闘中や戦闘後に**体力を回復**するための主要な手段です。 |
| **ウィンチェスター** | 銃砲店 | 高ダメージで正確なライフル。すべての初心者におすすめです。 |
| **ホイットニー・ドラゴン**| 銃砲店 | 連射速度と阻止能のバランスが良い、信頼できるサイドアームです。 |

リッジビー郡には**銀行（Bank）**もあり、武器やアイテムを保管できます。持ち運べる量には制限があるため、銀行を利用して、死亡時に失いたくないレアな戦利品を安全に保管しましょう。また、**理髪店**や**仕立て屋**を訪れて外見をカスタマイズすることもできますが、これらは純粋に装飾的なものです。

## スタンドと遺体パーツの入手方法

スタンドは、ユニークな能力を提供する超常的な具現化です。Bridger Westernでは、それらを入手する主な方法が2つあります。スタンドの矢（釣りから入手）と遺体パーツです。

### 遺体パーツのサイクル
一定時間ごとに、サーバー全体のメッセージで遺体パーツがスポーンしたことが告知されます。空に光の柱が現れ、その場所を示します。
*   **発光：** 遺体パーツを拾うと、キャラクターが白く光ります。マップ上のすべてのプレイヤーがこの光を見ることができるため、標的になります。
*   **融合：** パーツを保持している間、数秒ごとに一定の確率でパーツが自分と融合し、ランダムなスタンドが付与されます。
*   **ドロップ：** パーツを保持している間に殺されると、パーツはその場にドロップし、他の誰かが獲得できるようになります。

### スタンドティアの概要
ほとんどのスタンドは運次第で入手可能ですが、一部のスタンドは特定のアクションを必要とします。

*   **一般/レアスタンド：** スタープラチナ、ザ・ワールド、シルバーチャリオッツ、クレイジー・ダイヤモンドなど。
*   **D4C（いともたやすく行われるえげつない行為）：** 最も入手が困難なスタンドです。累計100キル、単一サーバーでの10連続キルが必要で、さらに遺体パーツがスポーンした際に砂漠で「ダウン（knocked）」状態になっている間に10%の確率を引き当てる必要があります。
*   **ホワイトスネイク：** 現在は管理者専用のスタンドであり、2026年時点では一般プレイヤーは入手できません。

## 魔女の小屋とカードビルド

XPを獲得して**ティア（Tier）**（Mメニューで確認可能）を上げると、カードスロットが解放されます。ティア3で最大3つのスロットを持つことができます。これらのスロットを埋めるには、沼地にいる魔女を訪ねる必要があります。

> **⚠️ 警告：** 加齢（Aging）は実際のメカニクスです。死ぬたびにキャラクターは年を取ります。高齢になりすぎると、ステータスにペナルティが発生する可能性があります。魔女の小屋で**ドッグベインのハーブ（Dog Bane herb）**（釣りやボスドロップで入手）を使用し、20歳若返らせましょう。

### 推奨カード組み合わせ
魔女に少額のMoolaを払って「運勢を占う」ことで、パッシブカードをロールできます。

| カード名 | 効果 | ビルドタイプ |
| :--- | :--- | :--- |
| **Sturdy（頑丈）** | 体力と姿勢（posture）が増加します。 | 防御 / タンク |
| **Quick Draw（早撃ち）** | 武器の取り出しとリロードが速くなります。 | ガンスリンガー / PvP |
| **Desperado（デスペラード）** | 低体力時にダメージが増加します。 | ハイリスク / ハイリターン |
| **High Noon（ハイヌーン）** | Ocular Prowessの命中精度を強化します。 | スナイパー / エイムアシスト |

人気の「メタ」ビルドの一つに、**High Noon**と**Evil Eye**の組み合わせがあります。この組み合わせはOcular Prowessの能力を大幅に強化し、セミオートエイムのような効果を生み出すため、初心者でも動く標的に当てやすくなります。

## フロンティアのための高度なヒント

*   **馬のテクニック：** 馬に乗って水の中を移動するときは、**Shift + Space**を押し続けてください。これにより、通常の泳ぎよりもはるかに速く、馬が水の中を「疾走」できるようになります。
*   **マルコム 70（Malcolm 70）：** ゲームに慣れてきたら、マルコム 70のためにMoolaを貯めましょう。これはスナイパーライフルで、カウボーイハットを被っていないプレイヤーなら一撃で倒せることも多いです。
*   **スポーン地点：** スポーン地点の設定を忘れずに！リッジビー郡のホテルや、砂漠にある様々なNPCキャンプで設定できます。これにより、敗北した後に何もない場所でスポーンするのを防げます。
*   **ギャングに参加する：** MメニューからギャングIDを入力します。グループでプレイすることは、遺体パーツを確保するための最良の方法です。パーツが融合する間、チームメイトがあなたを守ってくれるからです。

## よくある質問（FAQ）

**Q: Bridger Westernで手っ取り早くMoolaを稼ぐには？**

A: 初心者にとって最も効率的なのはフィッシング・ヴィレッジでの釣りです。インベントリいっぱいの魚を売れば、1,000から5,000 Moolaを稼げます。経験豊富なプレイヤーなら、山賊のボス狩りやトンネルでのPvPデュエルに勝利することも有効な選択肢です。

**Q: 初心者が使うのに最適なスタンドは何ですか？**

A: シルバーチャリオッツとスタープラチナは、分かりやすい近接コンボと高い防御性能を持っているため、初心者に最適です。遠距離を好むなら、ハイエロファントグリーンが優れたユーティリティを提供します。

**Q: キャラクターが白くなって光り出したのはなぜですか？**

A: それはあなたが遺体パーツを保持していることを意味します。スタンドを受け取るには、それが「融合」するまで生き残る必要があります。光っている間はマップ上の全員から見えるので注意してください。

**Q: このBridger Western初心者向けチュートリアルは2026年版に対応していますか？**

A: はい、このガイドは魔女の小屋のカードシステム、現在のスタンドラインナップ（タスクやD4Cを含む）、更新されたリッジビー郡のショップなど、最新のメカニクスをカバーしています。
