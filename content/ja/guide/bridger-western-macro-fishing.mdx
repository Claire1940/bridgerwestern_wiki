## マクロバージョンの理解：V1 vs. V2

インストールを開始する前に、自分のプレイスタイルに合った自動化レベルを決める必要があります。コミュニティでは一般的にスクリプトを2つの主要バージョンに分類しています。V1「ヘルパー」は、矢印や宝箱のミニゲームの煩わしさを避けたいアクティブなプレイヤー向けで、V2「完全放置（AFK）」バージョンは、PCを数時間動かし続けたいプレイヤー向けに構築されています。

| 機能 | V1（ミニゲームヘルパー） | V2（完全放置マクロ） |
| :--- | :--- | :--- |
| **自動キャスト** | いいえ（手動入力） | **はい（完全自動）** |
| **自動リール** | はい（検出時） | **はい（音の合図時）** |
| **ミニゲーム解決** | OCRベースのパターン | **OCRベースのパターン** |
| **餌の使用** | 手動 | **オプションで自動** |
| **主な用途** | ノックバック回避 | **一晩中のムーラ稼ぎ** |

> **⚠️ 警告:** Bridger: WESTERNでの自動化の使用は、厳密にはゲームの利用規約に違反しています。多くのプレイヤーが問題なくこれらのスクリプトを使用していますが、アカウント停止のリスクは常にあります。メインアカウントの進行状況が心配な場合は、サブアカウントを使用してください。

## 釣り自動化の必須要件

どのスクリプトを選択しても、2026年に **bridger western macro fishing** を機能させるためには、譲れない3つの要件があります。これらが一つでも誤設定されていると、マクロはミニゲームの文字を「認識」できなかったり、魚の食いつきを「聞き取る」ことができなかったりします。

1.  **AutoHotkey (AHK):** ほとんどのスクリプトはAutoHotkeyエンジンで構築されています。最新の2026年版がPCにインストールされていることを確認してください。
2.  **解像度スケール:** PCのディスプレイ設定は必ず **100%のスケーリング** に設定する必要があります。Windowsの設定が125%や150%になっていると、マクロがスキャンするピクセル座標がずれてしまいます。
3.  **Robloxグラフィックス:** 水面がクリアに見え、かつUI要素（文字）が鮮明に保たれるレベルにグラフィックスを設定してください。

## セミAFKマクロのセットアップ

PCの前にはいたいけれど、ミニゲームの「面倒な部分」が嫌いな人には、セミAFKマクロが最も安全な選択肢です。このバージョンでは、スクリプトが釣りのパターンの機械的な実行を処理している間、バックグラウンドでゲームをプレイすることができます。

### 設定手順
セミAFKバージョンを機能させるには、スキャン領域の特定の座標を設定する必要があります。これは通常、**O** キーと **P** キーを使用して、検出ボックスの上端と下端の境界を設定することで行います。

*   **ステップ1：** 水辺に近づき、釣り竿を装備します。
*   **ステップ2：** **O** キーを押して、ミニゲームUIが表示されるスキャン位置の上端を設定します。
*   **ステップ3：** **P** キーを押して、下端の境界を設定します。
*   **ステップ4：** **左ブラケット [** キーを押して、マクロをオンに切り替えます。

標準的な1920x1080の解像度の場合、多くのプレイヤーは上端を `920, 500` 付近、下端を `1, 580` 付近の座標で成功しています。ラグのためにマクロのクリックが速すぎてパターンを逃してしまう場合は、スクリプトの「Click Delay（クリック遅延）」の値を高い数値（例：50msから80ms）に編集してください。

## V2 完全放置マクロのマスター

V2マクロはより複雑なツールで、音声検出を利用して魚が針にかかったタイミングを判断します。これにより、スクリプトは人間の介入なしに自動的にリールを巻き、再キャストすることができます。これが2026年における **bridger western macro fishing** のゴールドスタンダードです。

### 音声の同期
V2マクロが失敗する最も一般的な理由は、ゲームの音を「聞き取れない」ことです。スクリプトは魚が食いついた際の水しぶき音をリッスンするため、音量設定を最大にする必要があります。

| 設定 | 必要な値 | 理由 |
| :--- | :--- | :--- |
| **Robloxマスター音量** | **100%** | 水しぶきの音がリールを起動させるのに十分な大きさであることを保証します。 |
| **Windowsシステム音量** | **100%** | OSがアプリケーションの音声を消音するのを防ぎます。 |
| **出力デバイス** | **一致させる** | マクロはRobloxが使用しているのと同じデバイスをリッスンする必要があります。 |
| **ゲーム内BGM** | **0% / オフ** | 音楽トラックが音声検出を妨げるのを防ぎます。 |

### 視覚的なキャリブレーション
ミニゲーム中にOCRが文字（W, A, S, D）を正しく識別できるように、カメラを完璧な位置に配置する必要があります。
1.  カメラを最大までズームインします（一人称視点）。
2.  正確に **7クリック分** ズームアウトします。
3.  F1オーバーレイ領域を、文字がスポーンする画面中央に直接合わせます。

> **💡 ヒント:** マクロが文字を見逃す場合は、設定メニューの「Tolerance（許容誤差）」設定を上げてください。Bridger: WESTERNで使用されている明るい白のテキストには、通常100の許容誤差が推奨されます。

## 高度な最適化とトラブルシューティング

完璧なセットアップでも、キャラクターのノックバックやキャストの失敗などの問題が発生することがあります。最新の2026年版マクロには、自動化をRobloxエンジンに対してより人間らしく見せるための、ランダム化されたスキャン遅延などの「アンチチート」対策が含まれています。

### よくある修正方法
*   **オーバーレイが表示されない:** 検出ウィンドウが消えた場合は、マクロフォルダ内の `settings.json` または `.ini` ファイルを削除して再起動してください。これにより、ウィンドウがメインモニターの中央にリセットされます。
*   **キャラクターが動く:** ミニゲームの物理演算によってキャラクターが押されている場合は、「Disable Mini-game（ミニゲームを無効化）」のトグルがアクティブであることを確認してください。ただし、これによりレアな宝箱が釣れなくなる可能性があることに注意してください。
*   **マルチモニター:** デュアルモニター設定の場合、マクロが間違った画面をスキャンしている可能性があります。設定タブを確認し、「Display Index」がRobloxを実行しているモニターと一致していることを確認してください。

| 問題 | 考えられる原因 | 解決策 |
| :--- | :--- | :--- |
| **マクロがリールを巻かない** | 音声の不一致 | システムとRobloxの音量を **100%** に設定してください。 |
| **違うキーが押される** | 解像度スケーリング | Windowsの表示スケールを **100%** に変更してください。 |
| **マクロがキャストに失敗する** | インベントリスロットのエラー | 釣り竿が **1番目のスロット** にあることを確認してください。 |

## 餌による利益の最大化
長期的な放置のためにV2マクロを使用している場合は、「Auto-Bait（自動餌）」機能を有効にする必要があります。ほとんどのスクリプトは、**インベントリの2番目のスロット** にある餌を探すようにプログラムされています。餌を使用するとヒット率が大幅に向上し、**bridger western macro fishing** プロセスの効率が高まります。餌がない場合、次のヒットまで最大30秒待つことがありますが、餌があればこれを10秒未満に短縮できます。

## FAQ

**Q: Bridger Westernのマクロ釣りスクリプトはBANされませんか？**

A: 100%安全な自動化はありません。Bridger: WESTERNの開発者は主にゲームを壊すようなエクスプロイトをターゲットにしていますが、マクロは技術的には禁止されています。リスクを最小限に抑えるため、メインドックのような人が多い場所で長時間釣りをすることは避けてください。

**Q: 数分後にマクロが止まってしまうのはなぜですか？**

A: これは通常、Robloxの「アイドル切断」または音声検出の失敗によるものです。システムの音量が100%であること、およびマクロ設定で「Keep Alive（キープアライブ）」または「Anti-AFK（放置防止）」機能が有効になっていることを確認してください。

**Q: マクロでレジェンダリーフィッシュを釣ることはできますか？**

A: はい。V2マクロは、レジェンダリーフィッシュに必要なパターンを解決する能力があります。ただし、「Click Delay」が高すぎると、マクロが最速で動くバーに追いつけない場合があります。

**Q: マクロはモバイルやMacで動作しますか？**

A: ほとんどの **bridger western macro fishing** ツールはAutoHotkeyを使用して.exeファイルとして開発されているため、Windows PC専用です。2026年現在、モバイルやmacOS向けの信頼できる完全自動釣りマクロは存在しません。
