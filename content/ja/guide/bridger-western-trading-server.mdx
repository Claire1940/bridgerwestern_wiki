## ランバーNPCと伐採の仕組み

トレードを成功させるための旅を始めるには、まず **ランバーNPC (Lumber NPC)** を見つける必要があります。このキャラクターは、メインの居住地の外れにある小さな家の中にいます。彼から斧を購入することが、トレードメタにおける最初の投資となります。斧を装備したら、トレードに必要なリソースを採取するために荒野へと向かいます。

伐採の主な場所は「森林 (Forest)」と「沼地 (Swamp)」の2つです。それぞれのエリアには異なるリスクと報酬があり、どちらを選ぶかは忍耐力と現在の目標によります。

### 森林 vs 沼地での採取

| 場所 | ドロップ率 | 伐採速度 | 推奨事項 |
| :--- | :--- | :--- | :--- |
| **森林** | **ロカ** のドロップ率が高い | **速い** | 素早いトレードに最適 |
| **沼地** | ロカと **ドッグ・ベイン・ハーブ** | **遅い** | 複数リソースの収集に最適 |

## ステップバイステップ：トレードの開始方法

ロカを入手し、ティア3であることを確認したら、以下の手順に従って他のプレイヤーとトレードを行います。

1.  **パートナーを見つける:** 相手のプレイヤーもティア3であることを確認してください。
2.  **ロカを装備する:** アイテムを手に持ちます。
3.  **プレイヤーをクリック:** トレードしたいプレイヤーに狙いを定めてクリックします。
4.  **プロンプトを確認:** 「本当にトレードしますか？ (Are you sure you want to trade?)」という通知が表示されます。
5.  **交換を完了:** 両方のプレイヤーが確認を承諾すると、トレードインターフェースが開きます。

**Bridger Western トレードサーバー** を探す際、プレイヤーは「トレードハブ」が自然に形成されている人口の多いインスタンスを探すことがよくあります。これらは通常、ランバーNPCの近くや中央の町の広場に位置しています。

## トレード可能なものの理解

Bridger: Westernのトレードシステムは、アイテムに加えてキャラクターのステータスにも影響を与えるという点でユニークです。トレードを行うと、ゲームはあなたの「年齢 (Age)」を同期（均一化）させます。現在の年齢は **Mキー** を押してキャラクターメニューを表示することで確認できます。これは、チームメイトとキャラクターの進行状況を同期させたいプレイヤーにとって特に便利です。

### スタンドと1%のスキンチャンス

**Bridger Western トレードサーバー** の経済において最も求められているのは、スタンドの交換です。最近のスキンの導入により、ゲーム内のほぼすべてのスタンドにビジュアルバリアントが存在するようになりました。

| スタンド名 | スキンの有無 | 特記事項 |
| :--- | :--- | :--- |
| **スタープラチナ** | あり | 市場価値が高い |
| **ザ・ワールド** | あり | 複数のバリアントが存在 |
| **ゴールド・エクスペリエンス・レクイエム** | **なし** | 現在スキン対象外 |
| **タスク** | **なし** | 現在スキン対象外 |

> **⚠️ 警告:** 現在スキンを持っていないスタンドをトレードする場合、転送中にそのスタンドが自動的にスキンを獲得する **1%の確率** があります。この「運」の要素により、レアなコスメティックを狙って「ロール（抽選）」するために、一般的なスタンドをトレードすることがプレイヤーの間で流行しています。

## ティア3への進行の重要性

トレードシステムがティア3に制限されていることは、いくら強調してもしすぎることはありません。多くの新規プレイヤーが何時間もかけてロカをファームした後に、他のプレイヤーをクリックできないことに気づきます。早くティア3に到達するには、地元の賞金稼ぎ（Bounties）の完了やワールドイベントへの参加に集中してください。これらのアクティビティから得られる経験値は単純な作業よりも大幅に高く、**Bridger Western トレードサーバー** の機能にずっと早くアクセスできるようになります。

## FAQ

**Q: なぜ他のプレイヤーとロカをトレードできないのですか？**

A: 最も一般的な理由は、あなたかトレード相手のどちらかが **ティア3** に到達していないことです。また、ロカを装備した状態でプレイヤーモデルを直接クリックしているか確認してください。

**Q: トレード中にすべてのスタンドがスキンを獲得しますか？**

A: いいえ。まだスキンを持っていないスタンドにスキンが付与される確率は **1%** のみです。さらに、**ゴールド・エクスペリエンス・レクイエム** や **タスク** のような特定のスタンドには、現在の2026年アップデートでスキンが用意されていません。

**Q: Bridger Western トレードサーバーを見つけるのに最適な場所はどこですか？**

A: ほとんどのプレイヤーは、メインタウンのランバーNPCの家の近くに集まっています。より組織的なトレードを行いたい場合は、多くのユーザーがコミュニティDiscordサーバーに参加し、ゲームインスタンスに入る前に特定のアイテム交換を調整しています。

**Q: ファームには森林よりも沼地の方が良いですか？**

A: ロカに関しては、伐採速度が速いため森林の方が優れています。しかし、沼地は **ドッグ・ベイン・ハーブ** を見つけることができる唯一の場所であるため、複数の種類のクラフト材料を必要とするプレイヤーには沼地の方が適しています。
