## スタープラチナ・ザ・ワールドの入手方法

通常の進行で獲得できる標準的なスタンドとは異なり、スタープラチナ・ザ・ワールドには特定の状況とかなりの運が必要です。最新の **bridger western アップデートノート**によると、まずベースとなるスタープラチナ（「スタープラチナ・プライム」とも呼ばれる）を所持している必要があります。

進化を誘発するには、自分自身または他のプレイヤーが発動した「時止め（Time Stop）」の中にいる必要があります。時間が止まっている間に、時止め移動を行うかGキーを押してください。試行するたびに進化が発生する確率は **1%** です。

| 必要条件 | 詳細 |

## SPTWのムーブセットと能力解説

**bridger western アップデートノート**では、SPTWは前身よりも一撃あたりのダメージがわずかに低いものの、スピードと実用性は遥かに優れていることが強調されています。これにより、連続して複数の攻撃を当てることに依存する「コンボ重視」のスタンドとなっています。

### 戦闘能力一覧表

| 技名 | キーバインド | 効果とダメージプロファイル |
| :--- | :--- | :--- |
| **スタンドM1** | 左クリック | 敵を**ヒットラグドール**状態にする方向パンチ。 |
| **スタンドラッシュ** | E | 高速の連打。ベースSPよりダメージは低いが、スピードステータスが高い。 |
| **ヘビーパンチ** | R | 強力な一撃。ベースSPのようなダッシュ効果はないが、安定したフィニッシャーとなる。 |
| **ベアリングショット** | T | スターフィンガーに代わる技。弾丸を放つ：**ヘッドショット45ダメージ** / **ボディ10ダメージ**。 |
| **ビートダウン** | V | 最後にターゲットを吹き飛ばす、シネマティックな掴みとパンチのコンボ。 |
| **時止め** | G | 時間を **2秒間** 停止させる。クールダウンが非常に短く、連発が可能。 |
| **スタンドガード** | F | スタミナバーを消費して、飛んでくる弾丸を反射し、爆発を防ぐ。 |

### 「ベアリングショット」のメタ
T能力であるベアリングショットは、**bridger western アップデートノート**における最も重要な変更点の一つです。近接距離が必要だったスターフィンガーとは異なり、ベアリングショットは中距離からのプレッシャーを可能にします。ヘッドショットを決めれば、相手の体力バーの半分近くを瞬時に削ることができます。しかし、実戦で弾丸を狙うのは難しいため、多くの上級プレイヤーは時止めを発動した直後に使用して確実にヒットさせることを推奨しています。

## ステータス比較：SP vs SPTW

**bridger western アップデートノート**を確認すると、開発者がSPTWを特化型のツールとして意図していることが分かります。すべてのカテゴリーにおいて直接的なアップグレードではなく、むしろプレイスタイルの転換です。

| ステータス | スタープラチナ（ベース） | スタープラチナ・ザ・ワールド |
| :--- | :--- | :--- |
| **攻撃速度** | A | **S (上昇)** |
| **素のダメージ** | A | B (減少) |
| **時止め持続時間** | 長い | **短い (2秒)** |
| **時止めクールダウン** | 標準 | **短い (連発可能)** |
| **機動力** | 高い | 非常に高い |

時止めの持続時間が短い（わずか2秒）ことはナーフ（弱体化）に見えるかもしれませんが、クールダウンが大幅に短縮されたことで、より頻繁に敵を妨害できるようになります。テンポの速いウェスタンの銃撃戦では、数秒ごとに時を止めて素早いベアリングショットとラッシュのコンボを叩き込む方が、一度の長い停止よりも効果的な場合が多いです。

## 2026年版 最適化キャラクタービルド

SPTWの高速キットを最大限に活用するには、キャラクターのパッシブを生存能力とスタミナ管理に集中させる必要があります。最新の **bridger western アップデートノート**に見られる現在のメタに基づくと、以下のビルドがハイステークスなデュエルで最高のパフォーマンスを発揮しています。

### 推奨パッシブ

1.  **Sturdy（頑丈）:** 最大体力を **15% 増加**させます。さらに重要なことに、ダウンした際にトドメを刺すのに少なくとも3ヒット必要になるため、蘇生されるための貴重な時間や敵が隙を見せる時間を稼げます。
2.  **Time Heals All Wounds（時はすべての傷を癒やす）:** パッシブな体力回復に不可欠です。SPTWユーザーは時止めを利用した「ヒット・アンド・ラン」を多用するため、このパッシブにより、位置を変えたり2秒間の停止のクールダウンを待ったりしている間に体力を回復できます。
3.  **Too Tired to Fall（倒れるには疲れすぎている）:** スタミナが50%を下回ると、受けるダメージが減少します。ダッシュやスタンド能力の使用でスタミナを激しく消耗するため、頻繁に「低スタミナ」状態になります。このパッシブはその疲労を防御バフに変えてくれます。

> **警告:** スタンドガード（F）は爆発を防ぐほど強力ですが、火炎ダメージは防げません。相手が炎系の能力を使用する場合は、スタンドに頼らず、優れたスピードを活かして射程外へダッシュしてください。

## 高度な戦闘戦略

**bridger western アップデートノート**をマスターするには、ボタンを知るだけでなく、戦闘の流れを理解する必要があります。SPTWの最も効果的なコンボは、「バーストと停止」のサイクルです。

*   **フリーズ・ショット・コンボ:** 時止め（G）を発動し、即座にベアリングショット（T）で頭を狙い、続けてラッシュ（E）を叩き込みます。時間が動き出す頃には、相手は甚大なダメージを受け、ラッシュによるラグドール状態に陥っています。
*   **防御的反射:** スタンドガード（F）を使って相手の弾丸を誘い出します。SPTWは弾丸を反射するため、ビートダウンの間合いに詰める前に、引き金を引きすぎるプレイヤーを彼ら自身の弾丸で倒せることがよくあります。
*   **スタミナ管理:** SPTWは高速なため、常にダッシュしたくなる誘惑に駆られます。スタミナバーを常に注視してください。スタミナが切れるとスタンドガードが失敗し、スタン状態になってしまい、次の銃撃に対して完全に無防備になります。
