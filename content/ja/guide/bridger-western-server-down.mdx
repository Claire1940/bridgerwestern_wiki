## トレードシステムとロカカカの種

サーバーが復旧したことで、経済は新しいロカカカシステムへと移行しました。トレードはもはや単純なメニュー操作ではなく、特定のアイテムとレベル要件が必要になります。

スタンドやカードをトレードするには、**ロカカカの実**を入手する必要があります。これらを入手するには、まず「木こりの斧」を手に入れ、森林（Forest）または沼地（Swamp）バイオームで木を伐採する必要があります。これにより、ゲームに「サバイバル」要素が加わり、単なる「釣りシミュレーター」からの脱却を図っています。

### トレードを成功させる方法
1.  **木こりの斧を入手する:** 新しい木こりNPCから購入します。
2.  **木を伐採する:** 沼地または森林へ向かいます。沼地の木からは稀に「Dog Vein」がドロップします。
3.  **種を入手する:** 木材を使用して、最終的にロカカカを確保します。
4.  **レベル3に到達する:** トレードを開始するには、両方のプレイヤーが少なくともティア3（レベル3）である必要があります。
5.  **分配:** ロカカカを持った状態で他のプレイヤーをクリックして、トレードを促します。

> **💡 ヒント:** トレード中に1%の確率で、スタンドのスキンが「シャイニー」になることがあります。これは現在、ゲーム内で最もレアなコスメティックティアです。

## Pay-to-Winの論争と新しいショップアイテム

**bridger western server down** の期間はコンソール対応など多くの要望された機能をもたらしましたが、同時にコミュニティを二分する「デベロッパープロダクト（開発者製品）」も導入されました。新しいUIの左上隅で、プレイヤーはRobuxを使用してさまざまな再抽選（リロール）を購入できるようになりました。

| 製品 | 効果 | コミュニティの反応 |
|---|---|---|
| **スタンド・リロール** | 5%の確率でスキンを入手 | 非常に物議を醸しており、「Pay-to-Win（課金優遇）」と見なされています。 |
| **馬のリロール** | 馬のステータスや外見を変更 | 一般的にコスメティック要素として受け入れられています。 |
| **顔/名前のリロール** | キャラクターの外見をカスタマイズ | 好意的。より良いロールプレイを可能にします。 |
| **眼の能力（Ocular Prowess）** | 目の色や能力をランダム化 | 賛否両論。ロールのレア度に依存します。 |

批判的な意見としては、スタンドのリロールを購入可能にすることは、特に「パッションプロジェクト（情熱を注いだプロジェクト）」と評されることの多いこのゲームにおいて、努力（グラインド）の価値を損なうというものがあります。しかし、開発者はこれらの製品が継続的なメンテナンスとサーバー費用の資金調達に役立つと主張しています。

## FAQ

**Q: なぜBridger Westernのサーバーはあんなに長くダウンしていたのですか？**

A: アップデート1を実装するために、サーバーは約4日間停止していました。これにはUIの全面的な刷新、トレードシステムの追加、XboxおよびPlayStationのコンソールサポートの統合が含まれていました。

**Q: 新しいアップデートでスタンドのスキンを入手するにはどうすればよいですか？**

A: 主に2つの方法があります。矢を使ってスタンドを引く際に5%の確率でスキンを入手するか、ロカカカの実を使用して他のプレイヤーとトレードする際に1%の確率で「シャイニー」スキンを入手することができます。

**Q: 私のゴールド・エクスペリエンス・レクイエム（GER）はどうなりましたか？**

A: GERは全面的なリワークのため、一時的にゲームから削除されました。所有していた場合は、レアなシャイニースタンドに置き換わっています。開発者は、GERが新しい要件とともに2026年の将来のアップデートで戻ってくることを確認しています。

**Q: 木こりNPCはどこにいますか？**

A: 木こりNPCは森林エリアの入り口付近にいます。木を伐採してロカカカの種を入手するために必要な斧を手に入れるには、彼と対話する必要があります。
