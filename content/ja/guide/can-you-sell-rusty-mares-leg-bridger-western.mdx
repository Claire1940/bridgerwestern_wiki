1.  **Ridge Countyへ移動する：** ここは武器のメンテナンスや特定のNPCクエストの主要な拠点です。
2.  **橋を見つける：** 中央の郡エリアから外へ続くメインの橋を渡ります。
3.  **銃職人NPCを見つける：** 橋の反対側に、小さな小屋や作業台の近くに立っているNPCがいます。
4.  **修理を開始する：** インベントリにRusty Mare's Legを入れた状態でNPCに話しかけます。
5.  **費用を支払う：** 修理を完了するには**1,500 moola**が必要です。

取引が完了すると、銃職人は錆びたアイテムを受け取り、修理された武器がストレージに移動したことを知らせてくれます。

## 銀行から武器を回収する

新しい武器を装備するには、**City Colony**（または銀行施設のある最寄りの主要な居住地）へ向かってください。銀行員に話しかけ、「Open Storage（ストレージを開く）」オプションを選択します。新しく作られたMare's Legがそこで待っています。このメニューから、アクティブなロードアウトに移動させることができます。

| 場所 | NPC | 目的 |

| **City Colony Bank** | 銀行員 | 保管された武器へのアクセス |
| **General Store** | 店主 | 弾薬や基本物資の購入 |

## Mare's Leg vs. Winchester：どちらが優れているか？

Mare's Legは、似たようなレバーアクションの美学を共有しているため、Winchesterライフルとしばしば比較されます。しかし、フィールドでの性能は大きく異なります。Mare's Legは実質的にライフルの「ソーンオフ（短銃身）」バージョンであり、ピストルと遠距離銃のハイブリッドのような存在です。

| 特徴 | Mare's Leg | Winchester |
| :--- | :--- | :--- |
| **連射速度** | **速い** | 普通 |
| **ダメージ** | 普通 | **高い** |
| **装弾数** | 少ない（6-8発） | **多い（10-12発）** |
| **射程距離** | 中距離 | **長距離** |
| **機動力** | **高い** | 普通 |

生の阻止能よりもスピードを重視する「ラン・アンド・ガン」のプレイスタイルを好むなら、Mare's Legは優れた選択肢です。しかし、遠距離戦や大型の獲物の狩猟には、依然としてWinchesterの方が優れた道具です。多くのプレイヤーは、その素早いドロータイムから、Mare's Legをセカンダリウェポンとして持ち歩くことを選んでいます。

> **ヒント：** Mare's Legは、相手が反応する前に素早く複数発撃ち込む必要がある町中での近接戦闘（CQC）で威力を発揮します。

## 銃職人を介したヴァンパイアクエストの解放

武器を修理してくれるNPCは、いくつかの儲かるクエストへの入り口でもあります。修理に費やした1,500 moolaを取り戻したいなら、彼のチャレンジを完了するのが最も効率的な方法です。

*   **射撃練習 I：** 缶に3回当てる。

## FAQ

**Q: Bridger WesternでRusty Mare's Legを他のプレイヤーに売ることはできますか？**

A: はい、ゲームの現在のトレードウィンドウがアクティブであれば、錆びたバージョンを他のプレイヤーと取引できます。しかし、多くのプレイヤーは1,500 moolaの修理費を節約するために、修理済みのものを買うか、自分で錆びたものを見つけることを好みます。

**Q: 1,500 moolaを持っていない場合はどうなりますか？**

A: 銃職人はアイテムを預かってくれますが、全額が支払われるまで修理プロセスは開始されません。狩猟、配達の仕事、または彼が提供する初期の射撃練習クエストを完了することで、すぐにお金を稼ぐことができます。

**Q: Mare's Legはメイン武器とセカンダリ武器のどちらと見なされますか？**

A: 2026年のメタでは、短銃身のため通常はセカンダリウェポンまたは「サイドアーム」としてスロットに入れられますが、ライフルタイプの弾薬を使用します。

**Q: 修理する前に使用すると、Rusty Mare's Legは壊れますか？**

A: 錆びたバージョンを実際に「使用」したり発砲したりすることはできません。銃職人が機能する状態に復元するまでは、厳密にインベントリアイテムとして機能します。

**Q: Vampire Ruinsはどこにありますか？**

A: Vampire Ruinsは、Ridge Countyから遠く離れた砂漠バイオームの奥深くに位置しています。初めてクエストに挑戦する場合は、十分な弾薬を持ち、少なくとももう一人のプレイヤーと一緒に挑むことをお勧めします。
