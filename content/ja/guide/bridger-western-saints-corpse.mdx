## Bridger Western 聖なる遺体 全出現場所

Bridger Westernの世界は広大であり、遺体のパーツは単に町の中心に現れるわけではありません。「聖人」が具現化する場所は7箇所あります。パーツが出現すると、画面上にグローバルメッセージが表示され、サーバー内のすべてのプレイヤーにとって目印となる巨大な光の柱が立ち昇ります。

### 1. 発掘現場 (The Dig Site)
町にあるメインブリッジの左側に位置し、遠くに見えるテントに向かって道を進みます。出現ポイントは発掘現場の最下部です。
> **警告：** ここは「死の罠」となる場所と考えられています。垂直の壁に囲まれているため、上からのスナイパーに対して無防備になり、脱出が困難です。脱出には登りロープを慎重に使用してください。

### 2. 線路沿いの道 (The Train Track Path)
町から、列車の進行方向とは逆（右側）に線路を辿ります。線路の左側に分岐する小さな道を探してください。このエリアは高台で広々としており、一番乗りしやすい場所ですが、出口が限られているため、危険なボトルネックでの小競り合いが発生しやすいです。

### 3. デッドバレーの岩山 (Death Valley - The Rock)
線路を辿って砂漠の奥深く、デッドバレー（死の谷）まで進みます。出現ポイントは、谷の壁沿いにある高い岩場の上にあります。
*   **アクセスのコツ：** よじ登り（マントル）ができない場合は、馬の上に立ち、武器（リボルバーやショットガン）を発射してキャラクターの高度をブーストし、棚に届くようにします。ここは町から遠く、下からの攻撃を防ぎやすいため、最高の防衛スポットの一つです。

### 4. 監視塔 (The Watchtower)
釣り場の近くで、山の方を向くと高い木造の監視塔が見えます。馬はこの地形を苦手とするため、徒歩で岩だらけの急斜面を進む必要があります。
*   **移動のコツ：** 塔のはしごや棚を登っている間は走らないでください。ゲームのよじ登りシステムの仕様で、端から吹き飛ばされ、登り直しになる可能性があります。

### 5. アントヒル島 (Ant Hill Island)
釣り小屋の右側の水域に位置します。ここはベテランの周回プレイヤーに非常に好まれるスポットです。主要な移動ルートから隔離されているため、多くのプレイヤーが見落としがちで、30秒ごとの出現判定を静かに待つことができます。

### 6. 沼地 (The Swamp)
アントヒル島と釣り小屋の真向かいにあります。遺体パーツは濁った水域の中央に出現します。光の柱は見つけやすいですが、遮蔽物のない沼地では「融合」プロセス中に格好の標的になってしまいます。

### 7. 草原の岩 (The Grassland Rock)
沼地の左側には広大な草原が広がっています。出現ポイントは特定の岩場の近くにあります。監視塔ほど特徴的ではありませんが、沼地や島に近いことから、この一帯はパトロールに最も効率的なエリアです。

## 出現メカニズムと確率の理解

**Bridger Westernの聖なる遺体**パーツは、ボスのようにな固定タイマーで出現するわけではありません。代わりに、ゲームは30秒ごとに「判定」を行います。この判定が成功するかどうかは、その時のサーバー内のプレイヤー数に完全に依存します。

開発者は、アクティブで人口の多いサーバーを促進するためにこのシステムを導入しました。プライベートサーバーや人数の少ないサーバーでプレイしている場合、確率は大幅に低下します。

| サーバー内の人数 | 出現確率（30秒ごと） |
| :--- | :--- |
| **3 - 5人** | 150分の1 |
| **満員サーバー (最大)** | **40分の1** |

「A Saint's Corpse has appeared!（聖なる遺体が出現した！）」というメッセージが点滅したら、レースの始まりです。光の柱に到達し、パーツにインタラクトして「融合（Merging）」プロセスを開始しなければなりません。

## 融合プロセスとスタンド報酬

**聖なる遺体**のパーツを拾っても、すぐに恒久的なアイテムとしてインベントリに追加されるわけではありません。代わりに、キャラクターがそれと「融合」し始めます。これはリズミカルな鼓動（パルス）によって表現されます。

> **重要な警告：** 融合の間、あなたは非常に無防備になります。融合が完了する前に殺されるとパーツをドロップし、他のプレイヤーに奪われる可能性があります。

融合時間はランダムです。最短で10秒、最長で2分かかることもあります。標準的なパーツが5つと、シークレットパーツが1つ存在します。

1.  **肋骨 (Ribcage)**
2.  **左腕 (Left Arm)**（10%の確率でタスク）
3.  **右腕 (Right Arm)**
4.  **左脚 (Left Leg)**
5.  **右脚 (Right Leg)**
6.  **心臓 (The Heart)**（D4C用のシークレットパーツ）

### スタンドレア度表
ほとんどのパーツは共通のスタンドをドロップする確率が等しいですが、最も切望されるスタンドは大幅に低い重み付けがされています。

| スタンド名 | レア度 | 必要パーツ |
| :--- | :--- | :--- |
| **タスク (Act 1)** | レア (10%) | 左腕のみ |
| **スタープラチナ** | ベリーレア | 任意の標準パーツ |
| **ザ・ワールド** | ベリーレア | 任意の標準パーツ |
| **ザ・ワールド AU** | ベリーレア | 任意の標準パーツ |
| **D4C** | クエスト報酬 | 心臓 |
| **標準スタンド** | コモン | 任意の標準パーツ |

**D4C (Dirty Deeds Done Dirt Cheap)** を入手するには、「心臓」パーツを含む特別なクエストラインを辿る必要があり、これは他の部位のようなランダムな出現よりも複雑です。

## 遺体周回のためのプロ戦略

公開サーバーで一人で**聖なる遺体**を狩ろうとすると、出現場所で待ち伏せる「スポーンキャンパー」や、楽しみのために狩りをする高レベルのスタンド使いに遭遇する可能性が高いです。2026年に効率を最大化するには、以下のコミュニティ戦略に従ってください。

### 「サーバー埋め」手法 (Server Fill Method)
1.  **低人数サーバーに参加：** プレイヤーが2〜3人のサーバーを探します。
2.  **グループを招待：** サーバーが満員になるまで、10〜12人の友人やDiscordメンバーを同時に参加させます。
3.  **順番リスト：** 次にパーツを受け取る人のリストを作成します。これにより内輪揉めを防ぎ、全員が最終的にスタンドを入手できるようにします。
4.  **保護サークル：** 特定の人物にパーツが出現したら、他の11人が融合のパルス中にその人の周りに境界線を作り、乱入してくるプレイヤーを撃退します。

### 環境の活用
**監視塔**や**デッドバレー**にいる場合は、高さを有利に利用してください。シャープスライフルなどの遠距離武器を装備しておきましょう。挑戦者があなたのポジションのふもとに到達する前に仕留めることができます。

### 部位の特定
実際にインタラクトするまで、どのパーツ（腕、脚、肋骨）が出現したかを知ることはできません。しかし、タスクをアンロックできるため、**左腕**が最も人気があります。目標がタスクである場合、運良く正しい部位が出るまで、何度か出現を繰り返す必要があるかもしれません。

## FAQ

**Q: 聖なる遺体はどのくらいの頻度で出現しますか？**

A: ゲームは30秒ごとに判定を行います。満員のサーバーでは、判定が成功する確率は40分の1です。これは平均して20分ごとにパーツが出現することを意味しますが、運（RNG）次第でそれよりずっと短くなったり長くなったりします。

**Q: 遺体パーツを他のプレイヤーとトレードできますか？**

A: いいえ。**聖なる遺体**にインタラクトした瞬間に融合プロセスが始まります。融合に成功すると、スタンドはあなたのキャラクターに紐付けられます。パーツをインベントリに入れて後でトレードすることはできません。

**Q: 融合中に切断された場合はどうなりますか？**

A: 鼓動（パルス）が発生している最中に切断されたりサーバーがクラッシュしたりすると、パーツは失われます。再ログインしたときにインベントリには入っておらず、おそらくサーバー内のどこかに再出現して他の誰かに見つけられることになります。

**Q: 左腕でタスクが出る10%の確率を上げる方法はありますか？**

A: 現在、左腕からのタスク出現率（10%）を上げるアイテムやゲームパスは存在しません。スタンドがアンロックされるまで、左腕を周回し続けるしかありません。
