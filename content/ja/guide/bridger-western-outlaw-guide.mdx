## 基本操作と移動

領土を主張する前に、ゲームの基本メカニクスを習得しなければなりません。Bridger Westernの動きは流動的ですが、障害物を利用したり敵の射撃を回避したりするには正確なタイミングが必要です。

| アクション | キーバインド | 説明 |
|---|---|---|
| **ダッシュ** | **W**を2回押し | 移動速度が上がり、障害物を自動的に乗り越えます。 |
| **回避（ロール）** | **Q** | 弾丸を避けたり、距離を詰めたりするための防御的な動作です。 |
| **射撃 / 照準** | **M1 / M2** | 左クリックで発砲、右クリックで精密な照準を合わせます。 |
| **リロード** | **R** | 小競り合いの間、常に銃を撃てる状態に保つために不可欠です。 |
| **オキュラー PRO** | **X** | ESPのような視覚を得て、敵をハイライトし命中精度を向上させます。 |
| **馬を呼ぶ** | **H** | 現在の馬を自分の位置に呼び寄せます。 |
| **乗馬 / 下馬** | **N** | 馬に乗り降りする際に使用します。 |
| **近接攻撃** | **G / C** | Gで武器による近接攻撃、Cで素手による格闘を行います。 |

> **⚠️ 警告:** 戦闘中のログアウト（戦闘中にゲームを離脱すること）は、キャラクターの年齢に多大なペナルティを与え、多くの場合10歳以上加齢してしまいます。

## アウトローの道を選ぶ

キャラクター作成時、アウトロー（無法者）かインロー（法執行者）のどちらかを選択することになります。2026年現在のほとんどのプレイヤーにとって、マップの移動やPvPにおける自由度が高いアウトローの陣営が好まれています。

効率的に旅を始めるには、**Red Corner（レッドコーナー）**でスポーンすることを目指してください。この場所は、初期の経済を支える主要な釣りスポットに戦略的に近い位置にあります。もしスポーン時の所持金が200〜300ムーラ未満であれば、キャラクターを「ワイプ（リセット）」してやり直すことを検討してください。最初の釣り竿と餌を購入するための初期資金は非常に重要です。

## 釣り経済：ムーラを素早く稼ぐ

釣りは、Bridger Westernにおいてレベルアップと通貨獲得のための最も信頼できる方法と広く認識されています。Reachby County（リーチビー郡）またはFishing Village（フィッシングビレッジ）に到着したら、**Taco Bait Shop（タコ・ベイトショップ）**に向かい、釣り竿といくつかの餌を購入しましょう。

### 効果的な釣りの手順
1. 水辺の近くで釣り竿を装備します。
2. ルアーに泡が出るのを待ちます。
3. ルアーが沈んだ瞬間に左クリックします。
4. 獲物を回収し、ベイトショップのNPCに売却します。

| アイテム / 魚 | 売却価格（目安） | 用途 |
|---|---|---|
| **タラ (Cod)** | 1,500 ムーラ | 素早く現金を稼げる高価値な魚。 |
| **バス (Bass)** | 500 ムーラ | ほとんどの水域で見つかる一般的な魚。 |
| **矢の破片 (Arrow Shard)** | N/A (伝説級) | 0.5%の確率で入手可能。スタンドを解放するために使用。 |
| **ドッグズベイン (Dog's Bane)** | N/A | キャラクターの年齢を下げるために使用される希少なハーブ。 |
| **宝箱 (Chests)** | 変動 | 武器、スチールワイヤー、爆発物が入っていることがあります。 |

## 馬の管理と高度なテクニック

馬は広大な西部の風景の中での命綱です。しかし、すべての馬が同じ性能を持っているわけではありません。初心者は基本的なラバから始めますが、1,000〜1,500ムーラ貯まったらすぐに上位の馬にアップグレードすることを目指すべきです。

### リロール戦略
Ridge County（リッジ郡）の駅近くにいる馬のNPCを訪ねてください。提示された馬が安い（800ムーラ未満）場合、速度やスタミナが低い可能性が高いです。「Show me another（他のを見せて）」オプションを使用して、価格が**1,100ムーラ以上**の馬が出るまでリロールしましょう。これらの伝説級の馬は、賞金稼ぎから逃げ切るために不可欠な優れたステータスを持っています。

### 水上移動テクニック
通常、馬は水中では遅くなりますが、「スペースバー・テクニック」を使うことでこれを回避できます。馬に乗ってダッシュしながら水に入るとき、**スペースバーを連打**してください。これにより、馬は川や沼地を高速で移動し続けることができ、魔女の小屋への到達や危険な遭遇からの脱出が容易になります。

## 進行状況：ティアとカード

Bridger Westernでは、プレイヤーのレベルは**ティア（Tier）**で表されます。最大ティア3まで到達でき、最大3つのパッシブ「カード」を装備できるようになります。これらのカードはビルドを決定づけ、超自然的な能力を授けることもあります。

カードを解放するには、沼地バイオームの奥深くにある**Witch's Hut（魔女の小屋）**へ行く必要があります。

1. **ティア1:** 最初のカードスロットを解放（初回は無料）。
2. **ティア2:** 2番目のカードスロットを解放（リロールに50ムーラ必要）。
3. **ティア3:** 最後のカードスロットを解放（リロールに50ムーラ必要）。

### 「エイムボット」ビルド
特定のカードを組み合わせることで、強力なシナジーを生み出すことができます。アウトローに最も好まれる組み合わせの一つは以下の通りです：
*   **High Noon（ハイヌーン）**
*   **Evil Eye（イービルアイ）**
*   **Folk's High Noon（フォークス・ハイヌーン）**

これらを**オキュラー PRO (X)**能力と併用すると、命中精度が大幅に向上し、コミュニティでは「合法エイムボット」と呼ばれるほどになります。

## スタンドと遺体の入手方法

スタンドはBridger Westernにおける究極のパワーアップです。入手方法は主に2つあります。釣りによる「遅くて安全なルート」か、遺体（Corpse Parts）を狩る「速くて危険なルート」です。

### 遺体の出現
「It appears once again（再び姿を現した）」という黄金の通知が表示されると、マップのどこかに**遺体（Corpse Part）**が出現し、巨大な黄金の光の柱が立ちます。

| サーバー人数 | 出現確率（30秒ごと） | 危険度 |
|---|---|---|
| **少人数 (1-5人)** | 150分の1 | 低 - 入手は容易だが、出現が稀。 |
| **中人数 (10人)** | 115分の1 | 中 - 1〜2人の競合を想定してください。 |
| **大人数 (25人)** | 40分の1 | 極限 - サーバー全員が奪い合いになります。 |

遺体を拾うと、キャラクターは数秒ごとにパルスを発します。各パルスには**15%〜20%の確率**でスタンドを付与するチャンスがあります。スタンドが完全に発現するまで、他のプレイヤーの猛攻を生き延びなければなりません。

## 装備と武器の場所

アウトローとして生き残るには、初期装備のピストルだけでは不十分です。武器の詳細なステータスについては、最新のバランス調整を確認するために[公式 Bridger Western Trello](https://trello.com/b/8G3R0X6A/bridger-western)を参照することをお勧めします。

*   **Ridge County Gun Shop（リッジ郡の銃砲店）:** 標準的なライフル、ウィンチェスター、マーベリック、トンプソン短機関銃を販売しています。
*   **Outlaw Hideout（アウトローの隠れ家）:** フィッシングビレッジの先にあります。ここのNPCから**コルト**や**ダブルバレル・ショットガン**を購入できます。
*   **弾薬パック:** どこの銃砲店でも購入できるほか、山賊の拠点でも見つかります。パックを左クリックして予備弾薬を補充しましょう。
*   **強壮剤 (Tonics):** 戦闘中の回復に不可欠です。常にインベントリに少なくとも3つは常備しておきましょう。

> **💡 ヒント:** 決闘中に弾切れになったら、**C**キーを押して拳に切り替えましょう。近接ダメージは意外と高く、リロード中の相手の不意を突くことができます。

## FAQ

**Q: Bridger Westernで年齢を下げるにはどうすればいいですか？**

A: 釣りや特定のバイオームの探索で見つかる希少なハーブ、**Dog's Bane（ドッグズベイン）**を摂取することで年齢を下げることができます。年齢を低く保つことは見た目の良さだけでなく、加齢による白髪やシワのエフェクトを避けることができます。

**Q: 素早くレベルアップするために、このガイドをどう活用するのがベストですか？**

A: 最速のレベルアップ（ティア上げ）方法は、継続的な釣りと、山賊の襲撃などのランダムなワールドイベントへの参加を組み合わせることです。タラのような高価な魚を売って最高級の装備を買うためのムーラを稼ぎ、それによってワールドイベントをより簡単にクリアできるようにします。

**Q: スタンドを入手した後、変更することはできますか？**

A: はい、可能ですが、キャラクターのスタンドを「リセット」する必要があります。これは通常、釣りで見つかる特定の植物を沼地の魔女のところに持っていくか、フルキャラクターワイプ（すべての進行状況をリセット）を行うことで可能です。

**Q: 馬の年齢は重要ですか？**

A: はい、馬が老化するとパフォーマンスがわずかに低下することがあります。馬が死んだ場合、リスポーンまで約2分かかります。長期的に最高のステータスを確保するために、早い段階で1,500ムーラの馬に投資するのが常に最善です。
