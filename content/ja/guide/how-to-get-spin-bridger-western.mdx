## スピンへの道：7つのステップ概要

旅を始める前に、スピンの習得はリニア（一本道）なクエストであることを理解しておく必要があります。最後のバトルまでスキップしたり、現金で能力を直接購入したりすることはできません。各目的は順番に完了する必要があり、キャラクターの年齢などの特定の要件が進行の大きな壁になることもあります。

| ステップ | 目的 | 主な要件 |
| :--- | :--- | :--- |
| **1** | **スピンの岩** | 光る渦巻き模様の岩に触れる |

## ステップ 1: スピンの岩に触れる

最初のタスクは「スピンの岩（Spin Rock）」を見つけることです。これらは特定の環境オブジェクトで、ランダムに黄金のエネルギーで「チャージ」されます。マップ上には6つの既知のスポーン地点があります。すべてを見つける必要はありません。現在アクティブな（渦巻きのエフェクトで光っている）ものを1つ見つけ、そこに触れるだけです。

| 場所 | 説明 |
| :--- | :--- |
| **キャニオン (Canyons)** | キャニオンバイオームの岩の露出部付近 |
| **発掘現場 (Dig Site)** | 発掘エリア近くの目立つ岩の上 |
| **町 (Town)** | メインの町を右に曲がった先。渦巻きは奥まった場所にあります |
| **湖 (The Lake)** | 水辺のすぐ隣にある岩の上 |
| **沼地 (Swamp)** | 魔女の小屋（Witch Hut）の裏。道をまっすぐ進みます |
| **アウトローの隠れ家 (Outlaw Hideout)** | 隠れ家の左側、上部の尾根沿い |

アクティブな岩が見つからない場合は、**サーバーホップ（サーバーの入り直し）**が最も効率的な方法です。既知の場所を確認し、光っていなければ、アクティブな岩が見つかるまで別のサーバーに参加してください。エネルギーに触れると画面に反応があり、最初のステップが完了したことを示します。

## ステップ 2: ジャイロ・ツェペリの居場所を突き止める

スピンのエネルギーに触れたら、次はジャイロを探す必要があります。岩と同様に、ジャイロにも6つの異なるスポーン場所があります。彼は放浪するNPCなので、木やサボテンに寄りかかっている彼を見つけるまで、いくつかのスポットを確認する必要があるかもしれません。

*   **キャニオンバイオーム:** 岩層の近くに立っていることが多いです。
*   **町の周辺:** 通常、町の左側にある大きなサボテンの隣にいます。
*   **町の中:** 町の境界内に稀に出現することがあります。
*   **リンゴの家:** 沼地の近くにある家の周囲の木々の近くを探してください。
*   **教会:** 沼地の奥深くにある教会の建物内にいます。
*   **アウトローの尾根:** アウトローの隠れ家近くにある家屋エリアの付近です。

彼に話しかけると、「拳を鍛える（toughen up your knuckles）」必要があると言われます。これがレッドコーナーへ向かう合図です。

## ステップ 3: レッドコーナーでの20キルストリーク

ここは多くのプレイヤーが行き詰まるポイントです。**レッドコーナー・ファイトクラブ（Red Corner Fight Club）**へ向かう必要があります。場所は、釣り桟橋から始まる線路を辿り、トンネルに到着するまで進みます。トンネル内にあるドアがクラブへと続いています。

このゾーン内で**20キルストリーク**を達成する必要があります。これは、もし19キルで死んでしまった場合、0からやり直さなければならないことを意味します。

> **💡 ヒント:** これを素早く終わらせるために、多くのプレイヤーはサブアカウントを使用したり、友人に協力してもらったりします。あなたの「標的」がアウトローであれば、死ぬたびにすぐトンネルに戻れるよう、アウトローキャンプにスポーン地点を設定してもらうと良いでしょう。

## ステップ 4 & 5: 決闘者の印とトゥルーマン

20キルストリークを達成したら、ジャイロの元へ戻ります。彼は次に「**決闘者の印（Duelist's Mark）**」を入手するよう命じます。これには、極めて低い確率でスポーンするNPC「**トゥルーマン（True Man / 真の男）**」を倒す必要があります。

トゥルーマンは、バンディット（強盗）やゾンビが通常出現する場所（リンゴの家など）ならどこにでもスポーンする可能性があります。彼はRNG（ランダム）ベースのスポーンであり、強制的に出現させる確実な方法はありません。プレイヤーによっては、一度の遭遇のために5時間以上待ったという報告もあります。

### トゥルーマンとの戦い方
トゥルーマンがスポーンすると、画面に短く「**すべてのツールを外せ（unequip all tools）**」というメッセージが表示されます。
1. **識別:** 彼は標準的なプレイヤーNPCのような姿をしていますが、不気味にこちらを凝視してきます。交戦しない場合、約30秒で消えてしまいます。
2. **戦闘ルール:** スタンドや特殊な武器を取り出すと、彼は即座に完璧な精度で射撃を開始します。遮蔽物を利用しながら、基本的な拳または標準的な銃を使用することをお勧めします。
3. **報酬:** 彼が死ぬ際、「ようこそ…『男の世界』へ（Welcome to a true man's world）」と言い、決闘者の印を受け取ることができます。

## ステップ 6 & 7: 最後の追い込み

決闘者の印を手に入れたら、最後にもう一度ジャイロの元へ戻ります。彼は十中八九、まだ「実戦経験が足りない」と言うでしょう。これはあなたの**累計キル数**を指しています。

*   **累計100キル:** キャラクターの履歴全体で合計100キルが必要です。これらはストリークである必要はありません。また、ファイトクラブでのキルは通常この累計にはカウントされないため、通常のワールドPvPに集中してください。
*   **現金 $25,000:** 能力を購入するための資金を用意しておく必要があります。

キル数と現金が揃ったら、ジャイロに話しかけてください。購入が完了しても能力が表示されない場合は、**キャラクターをリセット**してください。これでスピン（鉄球）が使用可能になります。

## スピンのメカニズムと能力

スピンをアンロックすると、[Bridger: Western](https://www.roblox.com/games/7255559124/Bridger-Western)のプレイ方法が根本的に変わります。標準的な銃を使用する能力は失われますが、代わりに2つの強力なスピンボール（鉄球）を手に入れます。

| 特徴 | 効果 |
| :--- | :--- |
| **銃の使用不能化** | スピンボールを相手に当てると、相手の火器を一時的に無効化します。 |
| **4つのモード** | スピンには4つの異なるモードがあり、飛び道具の物理挙動や動きを変化させます。 |
| **タスクとの相乗効果** | タスク（Tusk）を使用しながらスピンを持つことは**可能**です。これらは相互排他的ではありません。 |
| **永続的な選択** | 現在、一度習得したスピンを「削除」する方法はありません。 |

## FAQ

**Q: タスク（Tusk）のようなスタンドをすでに持っている場合でもスピンを入手できますか？**

A: はい。多くのプレイヤーはどちらか片方しか持てないと考えていますが、タスクを維持したままスピンの技術を使用することは完全に可能です。これらはハイレベルなPvPにおいて非常に相性が良いです。

**Q: トゥルーマンとの戦いで死んだらどうなりますか？**

A: 決闘中に死ぬと、そのステップは失敗となります。再びRNGによって彼がスポーンするのを待つ必要があり、これには数時間かかることもあります。回復アイテムを持参し、遮蔽物を利用することを強くお勧めします。

**Q: 最終ステップの100キルはどうやって確認すればいいですか？**

A: ゲーム内に「実戦経験」の100キルを具体的に追跡するUIはありません。手動でカウントするのが最善の方法です。ファイトクラブのストリークでの20キルは、通常この最終カウントには含まれないことを覚えておいてください。

**Q: Bridger: Westernでスピンをもっと早く入手する方法はありますか？**

A: **Bridger: Westernでスピンを早く入手する**最善の方法は、キルストリークと累計キルの要件にサブアカウントを使用し、スピンの岩とジャイロのスポーンを確認するためにサーバーホップすることです。トゥルーマンのスポーンだけは、プロセスの中で大幅にスピードアップさせることができない唯一の要素です。
