## 聖なる遺体：仕組みと出現率

聖なる遺体のパーツは、Bridger: Westernにおいて最も切望されるアイテムです。これらのアーティファクトは、スタープラチナ、ザ・ワールド、タスクといった上位ティアのスタンドを入手するための主要な手段です。これらのパーツがどのように出現するかの仕組みを理解することが、攻略への第一歩です。

遺体パーツは30秒ごとに出現する可能性がありますが、その確率はサーバー内に現在アクティブなプレイヤーの数に大きく影響されます。この仕組みはソーシャルなプレイを促す一方で、プライベートな環境でのファーミングを難しくしています。

| サーバー人数 | 出現確率（30秒ごと） |
| :--- | :--- |
| **3人以上** | 150分の1 |
| **満員のサーバー** | **40分の1** |

パーツが出現すると、画面上にグローバルメッセージが表示され、空に向かって巨大な光の柱が立ち上がります。これはパーツを回収できるプレイヤーは1人だけであるため、その場所への競争が始まったことを意味します。

### 遺体パーツとスタンドの可能性
標準的な遺体パーツは5つあり、さらに1つの秘密のパーツが存在します。ほとんどのパーツはスタンドの抽選プールを共有していますが、「左腕」はスピン系のスタンド「タスク」を求めるプレイヤーにとって特別な意味を持ちます。

| 遺体パーツ | スタンド抽選プール | 特記事項 |
| :--- | :--- | :--- |
| **肋骨 (Ribcage)** | 標準プール (SP, TW, TWAU) | 一般的なビルドで需要が高い。 |
| **左腕 (Left Arm)** | 標準プール + **タスク** | **10%の確率**でタスクを付与。 |
| **右腕 (Right Arm)** | 標準プール | コンプリートを目指すなら必須。 |
| **左脚 (Left Leg)** | 標準プール | 標準的なレアリティ。 |
| **右脚 (Right Leg)** | 標準プール | 標準的なレアリティ。 |
| **心臓 (Heart)** | **D4Cのみ** | 複雑なクエストラインを持つ秘密のパーツ。 |

> **⚠️ 警告:** パーツを拾った後、「融合（merging）」プロセスを経る必要があります。これには10秒から2分かかります。この間、出現をリセットするためにあなたを殺そうとする他のプレイヤーに対して、非常に無防備な状態になります。

## 聖なる遺体の全出現場所

この **Bridger Western ファームガイド** を効果的に活用するには、7つの主要な出現場所を暗記する必要があります。到着する前に光の柱がどこを指しているかを知ることで、戦術的に大きな優位に立つことができます。

1.  **発掘現場（底部）(The Dig Site):** メインの町の橋の左側に位置します。ここは危険な「穴」のエリアです。ここでパーツが出現した場合、ロープを使って登らなければならず、スナイパーの格好の標的になります。
2.  **高台の道（線路）(The High Path):** 町の右側にある線路（列車の進行方向とは逆）に沿って進みます。小さな道が台地へと続いています。見晴らしは良いですが、脱出ルートが1つしかありません。
3.  **デスバレーの壁 (Death Valley Walls):** 線路を砂漠の奥深くまで進み、デスバレーに到達します。出現場所は高い岩壁の上にあります。
    *   **プロのコツ:** この場所に素早く到達するには、馬の上に立ち、銃を撃ってジャンプの高さを「ブースト」して棚に飛び乗ります。
4.  **監視塔 (The Watchtower):** 釣りエリアの近くにあります。山を登り、さらに木製の塔を登る必要があります。
    *   **プロのコツ:** 梯子を登っている間は走ったりスプリントしたりしないでください。横から足を踏み外して転落死する恐れがあります。
5.  **アントヒル島 (Ant Hill Island):** 釣り小屋の右側の水中にあります。多くのプレイヤーがこの島を無視するため、待機場所としては最も安全なスポットの一つです。
6.  **沼地中央 (The Swamp Center):** 釣り小屋の真向かいにあります。パーツは濁った水の中央に出現します。視認性は非常に高いですが、遮蔽物がほとんどなく、身をさらすことになります。
7.  **草原 (The Grasslands):** 沼地エリアの左側、大きな岩の塊の近くです。広々としたエリアで、しばしば遠距離からの撃ち合いに発展します。

## 最高効率のお金稼ぎ：釣りと侵略

スタンド狩りはエンドゲームですが、キャラクターを維持するためには「ムーラ（お金）」が必要です。2026年における最も効率的な **Bridger Western ファームガイド** の戦略は、発掘現場近くでの特定のループです。

### ステップ1：装備を整える
まず、村へ向かい（列車に乗るか、線路沿いに馬を走らせる）、釣り小屋を見つけます。**釣り竿 (Fishing Rod)** を購入しましょう。初期資金が足りない場合は、NPCの侵略（Invasions）が発生するまで交通量の多い場所で待機します。これらのNPCを倒して、手っ取り早く軍資金を稼ぎましょう。

### ステップ2：発掘現場の川ループ
発掘現場の近くにある川は、いくつかの理由から最適なファーミング場所です。
*   **魚の価値:** 魚を売ることで、安定した信頼できる収入が得られます。
*   **ハーブ収集:** 釣りをしている間、レアなハーブを見つけることがあります。これらは **魔女NPCに1つ2,500ムーラ** で売却できます。
*   **NPCの侵略:** このエリアでは頻繁にNPCの攻撃が発生します。これらの攻撃者を倒すと、釣りスポットを離れることなくボーナスXPとお金が手に入ります。
*   **弾薬へのアクセス:** 発掘現場は馬ですぐの距離にあり、侵略が激しくなったときに素早く弾薬を補充できます。

| アクティビティ | 報酬タイプ | メリット |
| :--- | :--- | :--- |
| **釣り** | お金 / スタンドの矢 | 時々「スタンドの矢」が手に入るチャンスがある。 |
| **ハーブ売却** | **2,500ムーラ** | 単一アイテムでの最高売却額。 |
| **NPCキル** | XP / お金 | カードティアのレベルアップに不可欠。 |

> **💡 ヒント:** 釣りは **スタンドの矢** を入手するための隠れた方法でもあります。聖なる遺体を目指して戦う準備ができていない場合、釣りは「ザ・ワールド」のようなまともなスタンドを手に入れるための最良の代替手段です。

## 戦略的なグループファーミング

[Bridger: Western](https://www.roblox.com/) でのソロファーミングは、聖なる遺体を巡る競争が激しいため困難です。最も成功しているプレイヤーは、コミュニティのDiscordサーバーを通じて組織された「遺体狩り」を利用しています。

1.  **サーバー埋め:** 10人以上のプレイヤーのグループが人数の少ないサーバーに入り、サーバーが満員になるまで友人を招待します。これにより出現率を最大（40分の1）に高めます。
2.  **順番待ちシステム:** プレイヤーはリストを作成します（例：「1つ目のパーツはAさん、2つ目はBさん」）。
3.  **護衛部隊:** パーツが出現すると、グループ全員でその場所へ移動します。指定されたプレイヤーがパーツと融合している間、他のメンバーは周囲を警戒し、部外者や襲撃者（レイダー）を撃退します。

この調整された努力が、スタープラチナやD4Cのような上位ティアのスタンドをグループ全員が確実に手に入れるための最短ルートです。

## FAQ

**Q: XPを稼ぐためにこのガイドをどう活用するのがベストですか？**

A: 最も安定したXPは、釣りをしながらNPCの侵略を返り討ちにすることで得られます。これにより、お金を稼ぎ、スタンドの矢を探しながら、同時にカードティアのレベルを上げることができます。

**Q: 特定のスタンド「タスク (Tusk)」はどうやって入手しますか？**

A: タスクは聖なる遺体の **「左腕」** を使用することでのみ入手可能です。左腕を使用した際に10%の確率でタスクが付与されます。それ以外の場合は、標準のレアスタンドプールから抽選されます。

**Q: ハーブを一番高く売れる場所はどこですか？**

A: ハーブは魔女NPCのところに持っていきましょう。彼女は1つ2,500ムーラで買い取ってくれます。これは標準的な商人の価格よりも大幅に高い設定です。

**Q: どの遺体パーツが出現するか知る方法はありますか？**

A: いいえ、出現するパーツは完全にランダムです。それがどのパーツ（肋骨、腕、脚など）であるかは、拾い上げて融合が始まってから初めて判明します。
