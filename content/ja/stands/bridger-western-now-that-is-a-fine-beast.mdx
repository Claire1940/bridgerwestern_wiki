<Section icon={Target} title="「Fine Beast」のセリフと価格の解読">

**「bridger western now that is a fine beast」**というフレーズは単なるフレーバーテキストではありません。これは馬の合計ステータスプールを示す特定の指標です。『Bridger Western』では、個々のステータス（スピード、スタミナ、ジャンプ、ダメージ）を合算する計算式を使用して、NPCの描写的なセリフと最終的な価格を決定します。

NPCが「Fine Beast」や「Strong legs, clear eyes, a real beauty（強い足、澄んだ瞳、真の美しさ）」と言及した場合、その馬の合計ステータス値は現在のゲーム内最大値に近い可能性が高いです。これらの馬は通常、約**1,400から1,500通貨**で販売されます。高価ではありますが、対人戦や広大なマップの長距離移動において、最も安定したパフォーマンスを発揮します。

| セリフのキーワード | ステータスの意味 | 推奨アクション |
| :--- | :--- | :--- |
| **Fine Beast** | 合計ステータスが最大に近い | 資金があれば**即買い** |
| **Best Runner** | スピード重視 | 移動用に最適 |
| **Strong Legs** | スタミナ / ジャンプが高い | 山登りに適している |
| **Real Beauty** | 全体的なバランスが良い | 堅実なオールラウンダー |
| **Mid / Bad** | 合計ステータスが低い | **スキップ**してリロール |

<Callout type="warning" title="価格 vs レアリティ">
価格はステータスの*合計値*によって決まります。レアリティ（レジェンダリー/ミシカル）は、*個々のステータス*が1.25の上限を超えているかどうかで決まります。全体的なステータスは低いものの、ジャンプのステータスだけが異常に高い、安価なレジェンダリーの馬が見つかることもあります。
</Callout>

</Section>

<Section icon={Sparkles} title="レジェンダリーとミシカルのレアリティ解説">

『Bridger Western』の新規プレイヤーにとって最も混乱しやすいのが括弧（ブラケット）システムです。このゲームにおけるレアリティは、純粋に「ステータスの限界突破」に関するものです。すべての標準的な馬は、どのステータスにおいても**1.25**というハードキャップ（上限）を持っています。この制限を超えるには、レジェンダリーまたはミシカルのバリアントを引き当てる必要があります。
