## カードシステムの基本：パッシブを理解する

Bridger Westernにおいて「カード」はパッシブアビリティに相当します。ボタンを押す必要があるアクティブスキルとは異なり、これらのカードはキャラクターのステータスを強化する永続的なバフや、特定の条件下で発動するトリガーを提供します。通常、これらはゲームプレイを通じて獲得し、どれを装備するかによってあなたの「ビルド」が決まります。

現在の2026年のメタでは、カードは大きく4つのグループに分類されます：攻撃（銃/近接）、スタンド特化、防御、そしてユーティリティです。ダメージを与えるカードだけを積みたくなるかもしれませんが、長期戦においてはバランスの取れたアプローチの方が良い結果をもたらすことが多いです。

| カードカテゴリー | 主な焦点 | 推奨プレイヤー |
| :--- | :--- | :--- |
| **ガンスリンガー** | リロード速度、ダメージ、精度 | リボルバーおよびライフル使用者 |
| **ブローラー** | 近接速度と近接格闘の威力 | セイバーおよびシルバーダガー使用者 |
| **スタンド使い** | 機動力とスタンド固有の相互作用 | グリーン・ヘラファーおよびパイロットモード使用者 |
| **サバイバー** | 体力回復とダメージ軽減 | 全プレイヤー（ハイブリッドビルド） |

## Bridger Western スタンドパッシブガイド：スタンド性能の最適化

Bridger Westernのスタンドは強力な存在ですが、その潜在能力を最大限に引き出すにはキャラクターのパッシブに大きく依存します。現在、この**Bridger Western スタンドパッシブガイド**において最も注目すべき相互作用は、**グリーン・ヘラファー（Green Herafer）**（西部劇風のハイエロファント・グリーン）として知られるスタンドに関するものです。

スタンド使いにとって最も重要なカードは**ウィングド・マン（Winged Man）**です。多くのプレイヤーはこれが空中射撃専用だと思い込んでいますが、実際にはグリーン・ヘラファーにとって最高のパッシブです。スタンドのパイロットモードを使用している際、ウィングド・マンは飛行メカニクスを大幅に向上させ、地上にいる敵が攻撃を当てるのをほぼ不可能にします。

### 最強のスタンドサポートカード

| カード名 | 効果の概要 | シナジー |
| :--- | :--- | :--- |
| **ウィングド・マン** | 特定のモードで飛行能力を付与 | **グリーン・ヘラファー**のパイロットモードに必須 |
| **スターディ** | 全体的な姿勢維持と耐性を増加 | スタンド操作中の怯みを防止 |
| **シークレット・テクニック** | 独自の移動や回避手段を提供 | スタンドを解除した際の再配置に最適 |

> **💡 ヒント:** スタンド重視のビルドでプレイする場合は、必ず少なくとも1枚は防御カードを優先してください。スタンドがアクティブな間に本体が倒されると、即座に決闘に敗北します。

## 攻撃パッシブ：銃と近接

レミントンの冷たい鋼鉄やセイバーの鋭い刃を好むプレイヤーにとって、カードの選択肢はさらに広がります。攻撃カードは、精密なエイムとアグレッシブなプレイに報いるように設計されています。

### ガンスリンガー・ビルド
主に火器を使用する場合、目標はDPS（秒間ダメージ）を最大化し、リロード中のダウンタイムを最小限に抑えることです。「デスペラード」と「エクゼキューショナー」のカードがここでのゴールドスタンダードです。

*   **デスペラード（Desperado）:** 速射シークエンス中のダメージ出力を増加させます。
*   **エクゼキューショナー（Executioner）:** 低体力の敵に莫大なボーナスダメージを与えます。決闘のトドメに最適です。
*   **シルバー・プロディジー（Silver Prodigy）:** これは上級者向けのカードです。ダメージを大幅に強化しますが、このカードを機能させるにはショップで**銀の弾丸（Silver Bullets）を購入する**必要があります。
*   **クイックドロー（Quick Draw）:** 連射速度の速い武器には不可欠で、ダッシュや武器切り替え後に武器を構えるまでの時間を短縮します。

### 近接スペシャリスト
2026年の近接ビルドは、スピードと「粘り強さ」——相手に張り付き続ける能力——に依存しています。

*   **ボーイ・ウィズ・フィスツ（Boy with Fists）:** 近接攻撃の振りの速度と衝撃を増加させます。
*   **ショー・ミー・ア・グッド・タイム（Show Me a Good Time）:** アグレッシブな近接コンボに勢い（モメンタム）の増加で報いる特殊なカードです。
*   **ロングショット（Longshot / Longot）:** わずかなリーチのアドバンテージを提供し、敵の近接範囲のすぐ外側から攻撃を当てることができます。

## 防御と生存パッシブ

サバイバリティ（生存能力）に焦点を当てずして、**Bridger Western スタンドパッシブガイド**は完結しません。[RobloxのBridger Western](https://www.roblox.com/games/8553011401/bridger-WESTERN)のオープンワールドでは、数に圧倒されることがよくあります。防御カードは伏兵から生き残り、形勢を逆転させることを可能にします。

| 防御カード | 必要な理由 |
| :--- | :--- |
| **スターディ (Sturdy)** | スタンやノックバックを防ぐための最も信頼できるカード。 |
| **トゥー・タイアード・トゥ・フォール (Too Tired to Fall)** | 一撃死を免れることができる「土壇場」で役立つカード。 |
| **トニック・マスター (Tonic Master)** | 戦闘中に使用する回復アイテムの効果を高める。 |
| **タイム・ヒールズ・オール・ウーンズ (Time Heals All Wounds)** | 時間経過とともに、ゆっくりとパッシブで体力を回復させる。 |
| **スモール・カッツ (Small Cuts)** | 出血効果や軽微な飛び道具から受けるダメージを軽減する。 |

## 2026年推奨メタビルド

これらのカードをまとまりのあるビルドに組み合わせることが、フロンティアをマスターするための最終ステップです。この**Bridger Western スタンドパッシブガイド**の原則を利用した、コミュニティお墨付きの3つのビルドを紹介します。

### 1. 「止められない無法者」（銃特化）
このビルドは、生の火力と、ターゲットが回復する前に仕留めることに焦点を当てています。
*   **カード:** デスペラード、エクゼキューショナー、スターディ、クイックドロー
*   **武器:** デュアルリボルバーまたはレバーアクションライフル
*   **戦略:** クイックドローで先手を取り、デスペラードでHPを削り、エクゼキューショナーで確実に仕留めます。

### 2. 「エメラルド・パイロット」（スタンド特化）
空から支配したいグリーン・ヘラファー使いに最適化されています。
*   **カード:** ウィングド・マン、スターディ、タイム・ヒールズ・オール・ウーンズ、シークレット・テクニック
*   **スタンド:** グリーン・ヘラファー
*   **戦略:** パイロットモードを起動し、ウィングド・マンを使って敵の視界の外（上空）を維持しながら攻撃を浴びせます。

### 3. 「シルバー・デュエリスト」（近接/ハイブリッド）
シルバーダガーを楽しむプレイヤーのための、ハイリスク・ハイリターンのビルドです。
*   **カード:** シルバー・プロディジー、ボーイ・ウィズ・フィスツ、ショー・ミー・ア・グッド・タイム、トニック・マスター
*   **武器:** シルバーダガー / セイバー
*   **戦略:** シルバー・プロディジーを発動させるために、必ず銀の弾丸やアイテムを所持してください。スピードを活かして距離を詰め、トニック・マスターで戦線にとどまります。

> **⚠️ 警告:** 町を出る前に必ずインベントリを確認してください。シルバー・プロディジーのようなカードは、専用の弾薬を補充し忘れると全く役に立ちません！

## FAQ（よくある質問）

**Q: Bridger Westernで最強のカードはどれですか？**

A: ビルドによりますが、**スターディ（Sturdy）**が最も汎用性の高い最強カードと広く見なされています。銃撃戦でもスタンドバトルでも致命傷になりかねない「怯み」を簡単に起こさないようにしてくれます。

**Q: ウィングド・マンのカードはすべてのスタンドで機能しますか？**

A: 現在、この**Bridger Western スタンドパッシブガイド**におけるウィングド・マンの主な利点は、パイロットモード中の**グリーン・ヘラファー**に対するものです。他のほとんどのスタンドには、この特定のパッシブカードの恩恵を受ける飛行メカニクスがありません。

**Q: シルバー・プロディジーのカードはどうやって発動させますか？**

A: シルバー・プロディジーを機能させるには、銀の弾丸（Silver Ammunition）または銀の武器を装備している必要があります。このカードは銀ベースのダメージの倍率として機能するため、標準的な鉛の弾丸を使用している場合は、このカードの恩恵はゼロになります。

**Q: 防御カードを複数重ねることはできますか？**

A: はい、可能です。人気の「タンク」ビルドでは、**スターディ**、**トゥー・タイアード・トゥ・フォール**、**タイム・ヒールズ・オール・ウーンズ**を併用します。これにより非常に倒されにくくなりますが、攻撃ビルドに比べるとダメージ出力は低下します。
