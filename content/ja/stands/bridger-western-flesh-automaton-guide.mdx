## Flesh Automaton カードとは？

Flesh Automatonは、死を回避するメカニズムとして機能する **レジェンダリー（Legendary）** レアリティのカードです。Bridger Westernのほとんどの戦闘シナリオでは、体力を失うとダウン状態になり、トドメの一撃に対して無防備になります。しかし、Flesh Automatonを装備していれば、体力がクリティカルな閾値まで低下した瞬間に、「歩く肉体爆弾（Walking Flesh Bomb）」と呼ばれる特殊な状態に突入します。

このフェーズ中、キャラクターは不気味なノイズを発し始め、周囲の全員に爆発が差し迫っていることを知らせます。数秒の短いディレイの後、キャラクターが爆発し、敵のグループを一瞬で全滅させるほどの巨大な範囲（AoE）ダメージを与えます。

| 特徴 | 詳細 |
| :--- | :--- |
| **レア度** | **レジェンダリー** |
| **発動条件** | 体力がほぼゼロになる |
| **主な効果** | 歩く爆弾への変身 |
| **ダメージタイプ** | 大規模な範囲爆発（AoE） |
| **無敵特性** | リボルバーとライフルを無効化 |
| **持続時間** | 爆発までの数秒間 |

> **警告:** 爆発は強力ですが、予備動作フェーズ中に完全に「不死身」になるわけではありません。ショットガンのような高バースト武器は、爆発する前にあなたの冒険を終わらせる可能性があります。

## Flesh Automaton の入手方法

このレジェンダリーカードを入手するには、根気と「ムーラ（moola）」（ゲーム内通貨）の両方が必要です。レジェンダリーのドロップであるため、最初の試行で引き当てる確率は低いです。新しいカードを引くには、特定のNPCを訪ねる必要があります。

### 場所とコスト
Flesh Automatonを手に入れるには、**見捨てられた町（Abandoned Town）**へ向かい、カードディーラー（Card Dealer）を見つける必要があります。あるいは、NPCの **Bridger** と話すことでも可能です。レジェンダリーカードの抽選には、1回につき **450ムーラ** かかります。

1.  **ムーラを稼ぐ:** 釣り堀（タコベイトショップの近く）で釣りをし、複数回の抽選に必要な数千ムーラを素早く貯めます。
2.  **ディーラーを見つける:** 見捨てられた町へ行くか、Bridgerを探します。
3.  **カードを引く:** 1回につき450ムーラを支払います。
4.  **レア度を確認:** レジェンダリー特有の輝きを探し、Flesh Automatonを引けたか確認します。

## 戦闘メカニズム：無敵状態と弱点

Flesh Automatonが提供する最大の利点は、「爆弾」状態にある間に付与される特殊な無敵状態です。これにより、標準的な遠距離武器で狙い撃たれる心配をせずに、攻撃者に直接突撃することができます。ただし、この無敵は絶対的なものではありません。

### ダメージ耐性表
Flesh Automatonがアクティブな間に、何がダメージを与え、何が与えないかを理解することは生存に不可欠です。

| 武器/アビリティタイプ | ダメージを受けるか？ | 効果 |
| :--- | :--- | :--- |
| **リボルバー** | **いいえ** | 完全にブロック。ダメージもノックバックもなし。 |
| **ライフル** | **いいえ** | 弾丸が通り抜けるか、ダメージがゼロになる。 |
| **ショットガン** | **はい** | 爆発前にダウンまたは殺害される可能性がある。 |
| **吸血鬼の能力** | **はい** | カードの無敵フレームをバイパスする。 |
| **スタンド** | **はい** | スタンドによる攻撃は依然として高い脅威。 |

> **攻略のヒント:** 爆弾状態の時に相手がショットガンを構えたり、スタンドを召喚したりした場合は、移動キーを使ってジグザグに動きましょう。目標は、自動爆発がトリガーされるまでの数秒間を生き延びることです。

## 進行状況：カードスロットのためのレベルアップ

見つけたカードをすべて装備できるわけではありません。Bridger Westernでは、パッシブパワーを制限する「ティア（Tier）」システムを採用しています。Flesh Automatonのようなレジェンダリーカードを他のバフと併用するには、キャラクターをレベルアップさせる必要があります。

*   **ティア 1:** 最初のカードスロットが解放（無料）。
*   **ティア 2:** 2番目のスロットが解放（レベルアップと50ムーラが必要）。
*   **ティア 3:** 現在の最大ティア。3枚のカードをアクティブにできます。

これらのスロットを解放するには、沼地エリアにある **魔女の小屋（Witch's Hut）** を訪れる必要があります。リッジ郡（Ridge County）への線路を辿り、森林バイオームに入って小屋を探してください。魔女が「運勢を占う」ことで、カードを判明させ、装備してくれます。

| ティアレベル | 最大カード数 | 解放場所 | 判明コスト |
| :--- | :--- | :--- | :--- |
| **ティア 1** | 1 | 魔女の小屋 | 無料 |
| **ティア 2** | 2 | 魔女の小屋 | 50 ムーラ |
| **ティア 3** | 3 | 魔女の小屋 | 50 ムーラ |

## レジェンダリー抽選のためのムーラ稼ぎ

1回の抽選に450ムーラかかるため、信頼できる収入源が必要です。2026年において最も安定した方法は **「釣り」** です。タコベイトショップを訪れて釣り竿と餌を購入すれば、高額で売れる様々な魚を捕まえることができます。

*   **普通の魚:** 50〜100ムーラで売却。
*   **珍しい魚（例：U-Bats）:** 1スタックで900ムーラ以上で売れることもあります。
*   **特殊アイテム:** 釣りではスタンドの矢（0.5%の確率）や、キャラクターの年齢を管理するのに役立つドッグズベイン（Dog's Bane）のハーブも手に入ります。

約5,000ムーラ貯まったら、カードディーラーの元へ戻りましょう。この予算があれば約11回の抽選が可能で、レジェンダリーのFlesh Automatonを引き当てるチャンスが十分にあります。

## Flesh Automaton と他のカードのシナジー

真の脅威となるためには、クリティカルな体力状態になる前に、機動力や命中率を向上させるカードとFlesh Automatonを組み合わせるべきです。例えば、**Ocular Pro**（Xキーで発動）を強化するカードを使用すれば、最後の爆発が残った生存者を一掃するだけで済むほどのダメージを事前に与えることができます。

人気のシナジー：
*   **High Noon + Evil Eye:** Ocular Pro中に命中率が向上し、疑似的なエイムボット効果を付与します。
*   **スピード系カード:** Flesh Automatonの「歩く爆弾」状態が発動した後、敵との距離をより素早く詰めるのに役立ちます。

## FAQ

**Q: Flesh Automatonはスタンドに対して有効ですか？**

A: いいえ。このカードは標準的なリボルバーやライフルに対しては無敵になりますが、スタンドの能力や攻撃は、歩く爆弾状態のあなたにダメージを与え、倒すことができます。

**Q: Flesh Automatonのカードを引くのに最適な場所はどこですか？**

A: 最も一般的な場所は、見捨てられた町（Abandoned Town）のカードディーラー、またはBridgerという名前のNPCに直接話しかけることです。1回につき少なくとも450ムーラ用意しておきましょう。

**Q: 爆発が始まったら止めることはできますか？**

A: いいえ。体力が発動ポイントに達し、不気味な音が鳴り始めたら、タイマーが切れる前に敵がショットガン、吸血鬼の能力、またはスタンドであなたを殺さない限り、爆発は避けられません。

**Q: Flesh Automatonを他のカードと一緒に使うために、カードスロットを増やすにはどうすればいいですか？**

A: レベルを上げ、沼地にある魔女の小屋（Witch's Hut）を訪れてティアを上げる必要があります。ティア3で最大3枚のカードを装備できます。
