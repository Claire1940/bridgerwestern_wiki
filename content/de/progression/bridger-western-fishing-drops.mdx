### Die Drop-Mechanik verstehen
Wenn Sie erfolgreich angeln, erhalten Sie möglicherweise eine "Minispiel"-Belohnung. Dies ist nicht immer ein Fisch; es kann eine von drei Möglichkeiten sein: eine **Kiste**, ein **Mar's Leg** oder ein **Pfeilsplitter**.

**Mögliche Angelhiebe**

| Drop-Typ        | Beschreibung                                                         |
| :-------------- | :------------------------------------------------------------------- |
| **Fisch**       | Kabeljau, Barsch, Snapper oder ein seltener Goldener Barsch. Alle werden für Moola verkauft. |
| **Kiste**       | Enthält verschiedene Gegenstände, darunter Moola, Verbrauchsgüter und seltene Gegenstände. |
| **Mar's Leg**   | Ein rostiges Waffenteil, das zu einer mächtigen Schusswaffe verarbeitet werden kann. |
| **Pfeilsplitter** | Der seltenste und begehrteste Drop, der zum Erhalt von Ständen verwendet wird. |

### Kisteninhalte: Verborgene Schätze freischalten
Kisten, die beim Angeln erhalten werden, sind wie kleine Jackpots. Sie können eine Vielzahl nützlicher Gegenstände enthalten, aber einer sticht besonders hervor.

**Mögliche Kisten-Drops**

| Gegenstandskategorie | Spezifische Gegenstände                                      | Hauptnutzen                                                                                  |
| :------------------- | :----------------------------------------------------------- | :------------------------------------------------------------------------------------------- |
| **Währung**          | Moola                                                        | Direkte Geldspritze.                                                                         |
| **Verbrauchsgüter**  | Dynamit, Molotows, Munitionspakete                          | Nützlich für Kampf und Erkundung.                                                            |
| **Seltener Gegenstand** | **Hundebane**                                                | Setzt das Alter um 20 Jahre zurück, setzt den Stand zurück oder verkauft sich für 2,5k Moola an die Hexe. |

**Der Hundebane erklärt:**
Der Hundebane ist ein außergewöhnlich wertvoller Gegenstand aus Kisten. Seine Vielseitigkeit macht ihn sehr begehrt:
- **Altersrücksetzung**: Reduziert das Alter Ihres Charakters um 20 Jahre (Mindestalter 20).
- **Stand-Rücksetzung**: Ermöglicht es Ihnen, Ihren aktuellen Stand zurückzusetzen, nützlich, wenn Sie einen neuen ausprobieren möchten oder bereits einen Stand haben und ihn für ein Leichenteil freimachen müssen.
- **Hoher Verkaufswert**: Kann für beträchtliche **2,5k Moola** an die Hexe verkauft werden, was ihn zu einer ausgezeichneten Quelle für schnelles Geld macht.

### Mar's Leg: Eine Waffe der Gelegenheit

### Wichtige Angel-Tipps

| Strategie                      | Vorteil                                                                     | Einfluss auf Drops                                                              |
| :----------------------------- | :-------------------------------------------------------------------------- | :------------------------------------------------------------------------------ |
| **Konsequenter Ködereinsatz** | Verdoppelt die Partikelgenerierungsgeschwindigkeit, reduziert Wartezeiten. | Erhöht die Gesamtzahl der Drops pro Stunde erheblich.                           |
| **Dedizierte Angelsitzungen** | Konzentrieren Sie sich ausschließlich auf das Angeln ohne Ablenkungen.     | Maximiert die aktive Angelzeit und das Potenzial für seltene Drops.             |
| **Inventarverwaltung**         | Halten Sie genügend Platz für Fische, Kisten und seltene Gegenstände bereit. | Verhindert den Verlust wertvoller Drops aufgrund eines vollen Inventars.        |
| **Regelmäßiges Verkaufen**     | Verkaufen Sie regelmäßig gewöhnliche Fische und Moola-generierende Gegenstände. | Hält den Moola-Fluss aufrecht, finanziert weitere Köderkäufe und vermeidet Inventarüberladung. |
| **Neue Spots erkunden**        | Während Docks beliebt sind, bieten weniger überfüllte Gebiete möglicherweise andere Erfahrungen. | Potenziell weniger Konkurrenz um Angelknoten, was zu konstanten Fängen führt. |

### Wann angeln
Es gibt keine spezifische "beste Zeit" des Tages im Spiel zum Angeln, aber Angeln während ruhigerer Serverzeiten kann weniger Konkurrenz um Angelplätze bedeuten. Letztendlich ist Konsistenz wichtiger als Timing.

**Wichtige Erkenntnisse für effizientes Angeln**
- **Verwenden Sie immer Köder.** Dies kann nicht genug betont werden; es ist der wichtigste Faktor.
- **Seien Sie geduldig.** Seltene Drops sind aus gutem Grund selten. Angeln Sie weiter, und sie werden schließlich erscheinen.
- **Priorisieren Sie Pfeilsplitter**, wenn Sie einen Stand suchen. Angeln ist die sicherste Methode.
- **Unterschätzen Sie Moola nicht.** Der Verkauf gewöhnlicher Fische bietet eine zuverlässige Einnahmequelle.
