## Den optimalen Angelplatz wählen

Bevor Sie Ihr Skript starten, ist der physische Standort Ihres Charakters der kritischste Faktor für den Erfolg. Während sich viele Spieler an den Docks versammeln, sind diese Bereiche hochfrequentierte Zonen, die anfällig für Störungen durch andere Spieler und "Griefing" sind. Für ein stabiles Erlebnis empfehlen wir, abgelegenere Orte aufzusuchen, wie zum Beispiel den versteckten Wasserfall-Bereich.

| Merkmal | Empfohlene Einstellung | Grund |
| :--- | :--- | :--- |
| **Server-Population** | Niedrig (unter 5 Spielern) | Reduziert die Wahrscheinlichkeit von Störungen durch Spieler. |
| **Ping-Schwellenwert** | < 0,5ms (Waffentest) | Hohe Latenz kann dazu führen, dass das Makro Minigpiel-Eingabeaufforderungen verpasst. |
| **Standort** | Wasserfall oder abgelegener Bach | Verhindert, dass andere Spieler Ihren Charakter entdecken und töten. |
| **Ausrüstung** | Angel + Hochwertiger Köder | Köder ist optional, erhöht aber die Fangfrequenz erheblich. |

Um den Premium-Platz am Wasserfall zu erreichen, folgen Sie den Bahngleisen weg von der Hauptstadt. Indem Sie eine Schrotflinte oder einen Revolver benutzen, um sich das Gelände hinaufzuschießen ("Shotgun-Jump"), oder einen teleportationsfähigen Stand verwenden, gelangen Sie auf den Vorsprung hinter dem Wasserfall. Dieser Ort ist für Gelegenheitsspieler notorisch schwer zu erreichen und bietet einen natürlichen Schutz für Ihre automatisierte Sitzung.

> **💡 Tipp:** Führen Sie immer einen "Ping-Test" durch, indem Sie eine Waffe abfeuern. Wenn es eine spürbare Verzögerung zwischen Ihrem Klick und dem Mündungsfeuer gibt, wechseln Sie auf einen Server mit niedrigerer Latenz, um sicherzustellen, dass das Makro-Timing präzise bleibt.

## Konfiguration der AlphaForceBravo GUI

Nachdem die Software installiert ist, müssen Sie die Logik des Makros mit Ihrem In-Game-Setup abgleichen. Die GUI (grafische Benutzeroberfläche) ermöglicht es Ihnen, Hotkeys und Audio-Einstellungen anzupassen. Eine Besonderheit des **Bridger Angel-Makros** ist, dass es auf das Geräusch des Wasserklatschens "hört".

*   **Audio-Routing:** Der Bot hört auf Ihr Standard-Desktop-Audio. Vermeiden Sie es, Musik zu hören oder in einem lauten Discord-Call zu sein, während das Makro aktiv ist. Alternativ können Sie ein virtuelles Audiokabel verwenden, um nur den Ton des [offiziellen Roblox-Clients](https://www.roblox.com) an das Skript zu leiten.
*   **Hotkeys:** Legen Sie Ihre Angel und Ihren Köder auf bestimmte Slots (z. B. Slot 6 und 7). Aktualisieren Sie diese Nummern im Tab "Bot Settings".
*   **Visuelle Einstellungen:** Stellen Sie Ihre In-Game-Grafik auf 1. Dies minimiert die GPU-Belastung und verhindert Frame-Einbrüche, die das OCR-Scanning desynchronisieren könnten.

> **⚠️ Warnung:** Wenn das Makro versucht, das Minigpiel zu schnell abzuschließen, erhöhen Sie den "Mini-game Key Delay" in den erweiterten Einstellungen auf 0,1 oder höher.

## Kalibrierung und Pfadaufzeichnung

Der häufigste Grund für das Scheitern eines Makros ist eine falsche Kalibrierung. Sie müssen dem Skript genau sagen, wo es auf Ihrer spezifischen Monitorauflösung nach der Aufforderung "Drücke [Taste]" suchen soll.

### Der F3-Kalibrierungsprozess
1.  Beginnen Sie manuell zu angeln, bis ein Minigpiel erscheint.
2.  Drücken Sie sofort **F3**, um den Bildschirm einzufrieren.
3.  Ziehen Sie einen engen Rahmen um die Buchstabeneinblendung.
4.  Drücken Sie **Enter**, um die Koordinaten zu speichern.

### Bewegungs- und Anti-Makro-Pfade
Um die Anti-Makro-Erkennung des Spiels zu umgehen, muss das Skript Ihren Charakter gelegentlich bewegen. Dies geschieht über den "Ledge Path" (Vorsprung-Pfad) und den "Wander Path" (Wander-Pfad).

| Funktion | Hotkey | Erforderliche Aktion |
| :--- | :--- | :--- |
| **Box kalibrieren** | F3 | Zeichnen Sie eine Box um den Minigpiel-Buchstaben. |
| **Vorsprung-Pfad** | F5 | Nehmen Sie einen kurzen Weg von der Ecke bis zur Wasserkante auf. |
| **Wander-Pfad** | F6 | Nehmen Sie einen Weg auf, der Ihre Position zurücksetzt, falls ein Fisch verscheucht wurde. |
| **Start/Stopp** | F1 / F2 | Schaltet die gesamte Skriptausführung ein oder aus. |

Wenn Sie Pfade aufnehmen, starten Sie Ihren Charakter in einer Ecke. Wenn Sie an der rechten Wand stehen, stellen Sie sicher, dass "Right Wall" in der GUI ausgewählt ist. Dies stellt sicher, dass der Charakter, falls er angestoßen wird, automatisch zurück in die Ecke läuft, um seine "Heimatposition" zurückzusetzen.

## Erträge maximieren und Fail-Safe-Funktionen

Das Hauptziel bei der Verwendung eines **Bridger Angel-Makros** ist das Sammeln seltener Gegenstände. Im Jahr 2026 bleiben Pfeilscherben einer der wertvollsten Drops beim Angeln, und im Gegensatz zu bestimmten Kräutern haben sie kein einschränkendes Inventarlimit.

*   **Pfeilscherben:** Essentiell für hochrangiges Crafting und Handel.
*   **Geld farmen:** In der Regel können Sie innerhalb einer Stunde drei volle Stacks Fisch fangen, was je nach Fischart etwa 3.000 bis 9.000 $ einbringt.
*   **Fail-Safe-System:** Das AlphaForceBravo-Skript enthält eine "Five-Strike"-Regel. Wenn der Bot fünfmal hintereinander keinen Fisch fängt, führt er automatisch ein `Alt+F4` für das Spiel und sich selbst aus. Er speichert dann einen Screenshot des zuletzt gesehenen Frames und eine Logdatei, damit Sie analysieren können, warum das Makro gestoppt hat.

Um die höchste Fangrate zu gewährleisten, schauen Sie immer in einem 60-Grad-Winkel auf das Wasser. Das Skript verlässt sich gleichzeitig auf das visuelle Klatschen und das akustische Signal. Wenn Sie direkt auf Ihre Füße oder gerade auf den Horizont schauen, führt dies oft zu "Missed Catch"-Fehlern.

## FAQ

**Q: Warum klickt mein Bridger Angel-Makro während des Minigpiels nicht auf die Buchstaben?**

A: Dies liegt meist an einem falschen Tesseract-Pfad oder einer schlechten Kalibrierung. Stellen Sie sicher, dass Sie die F3-Taste verwendet haben, um eine Box um die Buchstabenaufforderung zu zeichnen, und dass Ihr "Mini-game Key Delay" nicht auf Null gesetzt ist. Überprüfen Sie auch, ob andere Fenster den Roblox-Client überlagern.

**Q: Kann ich das Makro verwenden, während das Spiel minimiert ist?**

A: Nein, die OCR-Technologie erfordert, dass das Spielfenster auf Ihrem Monitor sichtbar ist, damit sie den Bildschirm "sehen" kann. Sie können jedoch einen zweiten Monitor für andere Aufgaben verwenden, solange das Roblox-Fenster auf dem Hauptbildschirm aktiv und unverdeckt bleibt.

**Q: Besteht das Risiko, für die Verwendung eines Angel-Makros gebannt zu werden?**

A: Obwohl jede Automatisierung ein theoretisches Risiko birgt, verwendet das AlphaForceBravo-Skript menschenähnliche Bewegungspfade (F5/F6) und randomisierte Verzögerungen, um echtes Spielerverhalten nachzuahmen. Die Nutzung eines privaten oder schwach besiedelten Servers reduziert das Risiko von manuellen Meldungen durch andere Spieler zusätzlich.

**Q: Was soll ich tun, wenn das Makro bei Regen fehlschlägt?**

A: Der Regen in Bridger Western kann ziemlich laut sein, was die Audio-Erkennung stören kann. Wenn das Makro während Stürmen fehlschlägt, versuchen Sie, Ihre Gesamtlautstärke auf 100 % zu erhöhen und sicherzustellen, dass die "Template"-Version der OCR ausgewählt ist, da diese bei widrigen Wettereffekten im Allgemeinen stabiler ist.
