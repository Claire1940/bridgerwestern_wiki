## Wie man einem Server mit einem Code beitritt

2.  **Spielmenü öffnen:** Navigiere zum Tab "Server" auf der [Bridger Western Roblox Seite](https://www.roblox.com/games/5145773251/Bridger-Western).
3.  **Code eingeben:** Wenn das Spiel ein In-Game-Menü verwendet, suche im Hauptmenü nach dem Button "Privater Server" und füge deinen Code ein.
4.  **Bestätigen und Laden:** Stelle sicher, dass deine Roblox-Datenschutzeinstellungen es dir erlauben, privaten Servern beizutreten (auf "Jeder" oder "Freunde" eingestellt).
