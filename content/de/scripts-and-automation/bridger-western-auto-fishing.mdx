<Section icon={Settings} title="Installation und technische Einrichtung">

Um mit **Bridger Western Auto Fishing** zu beginnen, müssen Sie die richtigen Dateien von einer vertrauenswürdigen Community-Quelle beziehen. Die meisten zuverlässigen Makros für dieses Spiel werden auf GitHub gehostet und über spezielle Discord-Server verteilt, um sicherzustellen, dass Benutzer die neueste Version erhalten, die mit den Spiel-Engine-Updates von 2026 kompatibel ist.

<Steps>
  <Step num="1" title="Treten Sie dem Community Discord bei">
    Greifen Sie auf den offiziellen Makro-Discord-Server zu, der in seriösen Community-Foren bereitgestellt wird. Navigieren Sie zum Abschnitt 'Foren' und suchen Sie nach "Bridger", um den speziellen Makro-Thread zu finden.
  </Step>
  <Step num="2" title="Von GitHub herunterladen">
    Folgen Sie dem Link zum GitHub-Repository. Laden Sie immer die neueste Version (z. B. V1.2) herunter und suchen Sie speziell nach der `.exe`-Datei. Vermeiden Sie das Herunterladen von Quellcode, es sei denn, Sie möchten ihn selbst kompilieren.
  </Step>
  <Step num="3" title="Antivirus-Ausnahmen verwalten">
    Da diese Makros "Hooks" verwenden, um Ihren Bildschirm zu überwachen und Tastatureingaben zu simulieren, können Windows Defender oder andere Antivirus-Software sie als "Falsch-Positiv" kennzeichnen. Sie müssen den Makro-Ordner zu Ihrer Antivirus-Ausschlussliste hinzufügen, um ihn auszuführen.
  </Step>
  <Step num="4" title="Erste Ausführung">
    Führen Sie die `.exe` als Administrator aus. Wenn Windows eine "PC wurde geschützt"-Meldung anzeigt, klicken Sie auf 'Weitere Informationen' und dann auf 'Trotzdem ausführen', um den SmartScreen-Filter zu umgehen.
  </Step>
</Steps>

<Section icon={Target} title="In-Game Konfiguration und Ausrichtung">

Der häufigste Grund für das Versagen von **Bridger Western Auto Fishing** ist eine falsche Ausrichtung. Das "Auge" des Makros muss perfekt über dem Bereich zentriert sein, in dem das Spiel die Buchstaben anzeigt. Wenn Ihre Zoomstufe auch nur um eine Stufe daneben liegt, wird die OCR die Zeichen nicht erkennen können, was zu einem verpassten Fang führt.

<CardGrid cols={3}>
  <Card title="Zoom-Strategie">
    - **Schritt 1**: Zoomen Sie Ihre Kamera ganz hinein.
    - **Schritt 2**: Zoomen Sie genau 7 Mal heraus.
    - Dies erzeugt die Standard-UI-Skalierung, die das Makro erwartet.
  </Card>
  <Card title="Overlay-Platzierung">
    - Ziehen Sie das transparente Feld über die Mitte Ihres Bildschirms.
    - Stellen Sie sicher, dass es den Bereich abdeckt, in dem die Buchstaben "W, A, S, D" erscheinen.
    - Halten Sie das Overlay "Immer im Vordergrund" aktiviert.
  </Card>
  <Card title="Empfindlichkeitseinstellungen">
    - Passen Sie die **Toleranz** an, um die Beleuchtung zu berücksichtigen.
    - Eine höhere Toleranz erkennt mehr Farben, kann aber "Rauschen" verursachen.
    - Eine niedrigere Toleranz ist präziser, kann aber Buchstaben in Schatten übersehen.
  </Card>
</CardGrid>

<Callout type="success" title="OCR testen">
Verwenden Sie die Schaltfläche "OCR testen" im Makro-Menü, während das Angel-Minispiel aktiv ist. Dies zeigt Ihnen in einem kleinen Debug-Fenster genau an, was das Makro "sieht", und ermöglicht Ihnen, die Toleranzstufen fein abzustimmen.
</Callout>

| Parameter | Empfohlener Bereich | Auswirkung auf das Angeln |
| :--- | :--- | :--- |
| Toleranz | 50 - 120 | Höher = einfachere Erkennung in dunklem Wasser |
| Klickverzögerung | 15ms - 45ms | Niedriger = schnellere Reaktion, höher = menschlicher |
| Tastaturbelegung (Start) | F1 - F4 | Schaltet die Erkennung ein/aus |
| Zoomstufe | 7 Klicks heraus | Wesentlich für die UI-Ausrichtung |

</Section>

<Section icon={Shield} title="Risikomanagement und Fehlerbehebung">

Die Verwendung eines **Bridger Western Auto Fishing** Makros ist ein "auf eigene Gefahr"-Unterfangen. Da es sich nicht um einen vollautomatischen Bot handelt, ist es weniger wahrscheinlich, dass es von serverseitigen "Bewegungs"-Kontrollen erkannt wird, aber die visuelle Erkennung durch andere Spieler oder Moderatoren bleibt eine Bedrohung.

**Häufige Probleme und Lösungen:**
- **Fehlendes Overlay**: Wenn Sie das Overlay versehentlich vom Bildschirm ziehen, schließen Sie das Makro und löschen Sie die Datei `settings.json` im Ordner. Ein Neustart setzt das Overlay auf die Standard-Mittelposition zurück.
- **Ungenau Klicks**: Wenn das Makro die falschen Tasten drückt, ist Ihre "Toleranz" wahrscheinlich zu hoch (Erkennung von Hintergrundpixeln) oder zu niedrig (Buchstaben nicht klar erkennbar).
- **Spielabstürze**: Stellen Sie sicher, dass Sie keine anderen intensiven Hintergrundanwendungen ausführen, die die Bildschirmaufnahme stören.
