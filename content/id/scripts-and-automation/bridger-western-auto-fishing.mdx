Menggunakan makro **memancing otomatis Bridger Western** adalah upaya "gunakan dengan risiko Anda sendiri". Karena ini bukan bot yang sepenuhnya otomatis, kemungkinan terdeteksi oleh pemeriksaan "pergerakan" sisi server lebih kecil, tetapi deteksi visual oleh pemain lain atau moderator tetap menjadi ancaman.

**Masalah Umum dan Perbaikan:**
- **Overlay Hilang**: Jika Anda secara tidak sengaja menyeret overlay keluar layar, tutup makro dan hapus file `settings.json` di folder. Memulai ulang akan mengatur ulang overlay ke posisi tengah default.
