<Steps>
  <Step num="1" title="Unduh Perangkat Lunak Makro">
    Akses server Discord resmi (tertulis di panduan yang relevan) dan temukan bagian makro gratis. Cari "Bridger" dan unduh file EXE penyiapan dari tautan GitHub yang disediakan.   </Step>
  <Step num="3" title="Konfigurasi Pengaturan Dasar">

Pertimbangkan untuk mengaktifkan "Lewati mini game" untuk sengaja gagal dan melempar ulang jika Anda mau.

  <Step num="5" title="Posisi Lempar Kustom dan Umpan">
