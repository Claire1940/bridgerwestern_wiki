Untuk memulai pencarian Dogbane, pergilah ke **Righetti County**. Dari stasiun kereta api utama, Anda harus menavigasi ke arah area pesisir, khususnya ke **Tackle Bait Shop** (Toko Umpan Pancing).

### Persyaratan Memancing
Untuk mendapatkan ramuan tersebut, Anda harus berpartisipasi dalam minigame memancing. Kunjungi NPC di Tackle Bait Shop untuk membeli hal-hal berikut:

| Item | Biaya (Moolah) | Tujuan |

| **Umpan** | 15 | Meningkatkan tingkat tangkapan |
| **Ramuan Dogbane** | N/A | Drop langka dari memancing |

> **💡 Tips:** Ramuan Dogbane adalah drop langka. Mungkin diperlukan waktu beberapa menit atau bahkan satu jam memancing secara konsisten untuk menariknya dari air. Jangan berkecil hati jika Anda hanya mendapatkan ikan pada awalnya!
