## Mengoptimalkan Pengaturan Deteksi untuk Efisiensi Maksimal

Inti dari sesi **fishing macro bridger western** yang sukses terletak pada pengaturan deteksi. Makro ini menawarkan beberapa cara untuk "melihat" permainan, termasuk Pixel, NumPy, dan OpenCV. Meskipun deteksi Pixel cepat, metode **NumPy** secara luas dianggap yang paling stabil untuk sesi AFK jangka panjang di tahun 2026.

### Parameter Deteksi
Untuk memastikan makro tidak pernah melewatkan tangkapan, Anda harus menyesuaikan pengaturan berikut di tab konfigurasi:

1.  **Metode Deteksi:** Atur ke NumPy untuk stabilitas yang lebih baik di berbagai kondisi pencahayaan dalam game.
2.  **Jeda Pemindaian (Scan Delay):** Pengaturan `0.05` direkomendasikan. Ini menciptakan buffer kecil di antara pemindaian, mencegah CPU Anda terbebani secara berlebihan sambil tetap cukup cepat untuk mengenai setiap huruf.
3.  **Toleransi Piksel:** Ini menentukan seberapa banyak warna dapat bervariasi. Jika Anda memancing selama siklus siang-malam di Bridger Western, toleransi yang sedikit lebih tinggi membantu memperhitungkan bayangan.
4.  **Pemindaian Kosong untuk Keluar:** Atur ke `5`. Ini memberi tahu makro untuk menunggu lima pemindaian kosong sebelum menyimpulkan bahwa mini-game telah berakhir dan saatnya untuk melempar kembali.

| Pengaturan | Nilai yang Direkomendasikan | Tujuan |
|---|---|---|
| **Jeda Acak** | Diaktifkan | Meniru perilaku manusia untuk mengurangi risiko deteksi. |
| **Monitor Audio** | Aktif (untuk debugging) | Membantu Anda melihat apakah makro "mendengar" ikan menggigit. |
| **Perangkat Output** | Audio Sistem Default | Harus sesuai dengan perangkat tempat suara game Anda diputar. |

## Pemecahan Masalah Umum

Jika makro Anda tidak mengklik huruf atau gagal melempar kembali, penyebab paling umum adalah **Backend Management**. Jika status menunjukkan "Offline" atau "Connecting", Anda mungkin perlu mengklik tombol "Kill and Restart" di dalam antarmuka makro. Ini menyegarkan koneksi antara skrip dan jendela permainan.

Masalah umum lainnya adalah deteksi audio. Makro "mendengarkan" suara percikan saat ikan menggigit. Jika volume game Anda terlalu rendah atau jika perangkat output Anda diatur secara tidak benar dalam pengaturan makro, mini-game tidak akan pernah terpicu. Selalu maksimalkan volume dalam game Anda dan pastikan "Audio Monitor" makro menunjukkan aktivitas saat suara terjadi.
