## Kontrol Pergerakan dan Interaksi Penting

Menavigasi medan lembah yang kasar membutuhkan pergerakan yang lancar. Berbeda dengan game petualangan standar, pergerakan dalam simulator ini sangat terikat pada kondisi fisik dan stamina karakter Anda. Memahami cara beralih dari jalan santai ke lari cepat yang menyelamatkan nyawa sangatlah krusial saat peluru mulai beterbangan.

| Aksi | Keybinding | Deskripsi |
|---|---|---|
| **Dash Cepat** | Q | Sentakan gerakan singkat dan cepat untuk menghindari tembakan musuh. |
| **Lari Cepat** | W + W (Ketuk Ganda) | Meningkatkan kecepatan gerak dengan mengonsumsi stamina. |
| **Panggil Kuda** | H | Memanggil tunggangan pribadi Anda ke lokasi saat ini. |
| **Naik/Turun Kuda** | N | Tombol interaksi untuk naik atau turun dari kuda Anda. |
| **Menu Statistik** | M | Melihat kesehatan saat ini, bounty, dan statistik karakter. |
| **Tinju** | C | Menyiapkan tangan Anda untuk perkelahian atau interaksi non-mematikan. |

> **💡 Tip:** Jika Anda mendapati diri Anda tanpa kuda di tengah hutan belantara, gunakan tombol 'H' untuk melihat apakah tunggangan Anda berada dalam jangkauan. Jika kuda telah terbunuh, Anda mungkin perlu mengunjungi istal di salah satu dari dua kota utama untuk memulihkannya.

## Mekanik Pertempuran dan Senjata Api Tingkat Lanjut

Inti dari pengalaman ini terletak pada baku tembaknya yang realistis. **Kontrol bridger western** untuk persenjataan dirancang untuk meniru sifat manual senjata api abad ke-19. Anda tidak bisa sekadar menahan pelatuk; setiap tembakan harus disengaja, dan setiap pengisian ulang harus diatur waktunya dengan sempurna.

### Kontrol Senjata Utama

| Input | Aksi | Efek |
|---|---|---|
| **M1 (Klik Kiri)** | Tembak | Melepaskan peluru yang saat ini berada di dalam kamar peluru. |
| **M2 (Klik Kanan)** | Bidik | Beralih ke pandangan fokus di atas bahu untuk akurasi. |
| **R** | Isi Ulang | Memasukkan selongsong peluru baru ke dalam senjata secara manual. |
| **G** | Melee | Melakukan serangan popor senjata jika musuh terlalu dekat. |
| **X** | Ocular Prowess | Mengaktifkan indra yang dipertajam untuk melacak target atau ancaman. |

### Seni Quick Draw

Salah satu fitur menonjol dari sistem pertempuran adalah mekanik "Quick Draw". Untuk melakukannya, Anda harus menahan Klik Kanan (M2) untuk memposisikan tangan di atas sarung pistol. Melepaskan dan segera mengklik M1 memungkinkan Anda untuk melakukan "fanning" atau "hammering" pada revolver Anda, menembakkan peluru dalam suksesi cepat. Ini sangat efektif selama duel atau saat disergap oleh bandit.

## Duel dan Interaksi Sosial

Di Ridge B. Valley, pertempuran tidak selalu tentang baku tembak yang kacau. Game ini mendukung sistem duel formal. Dengan mengetik "duel" di chat atau menggunakan menu interaksi saat berhadapan dengan pemain lain, Anda dapat memulai tantangan formal.

Namun, berhati-hatilah: komunitas di sini sering kali paranoid. Mencabut senjata saat mengenakan topeng adalah sinyal universal untuk permusuhan. Jika Anda ingin tetap damai, simpan senjata Anda di sarungnya dan gunakan fungsi chat khusus untuk berkomunikasi.

> **⚠️ Peringatan:** Melakukan "tindakan tak terkatakan" terhadap pemain non-hostile akan menandai Anda sebagai Killer (Pembunuh). Begitu status ini aktif, toko dan penjahit tertentu, seperti Dutch dan Taylor Scurlock, mungkin menolak melayani Anda, mencegah Anda membeli pakaian atau perlengkapan baru.

## Berburu dan Bertahan Hidup di Alam Liar

Dunia meluas jauh melampaui jalanan berdebu di kota-kota utama. Pemain dapat menjelajah ke bioma salju untuk berburu sumber daya langka. Berburu membutuhkan penerapan **kontrol bridger western** yang berbeda, lebih berfokus pada siluman dan presisi daripada kekuatan tembakan mentah.

1.  **Cari Mangsa:** Gunakan 'X' (Ocular Prowess) untuk menemukan jejak atau tanda visual rusa dan beruang.
2.  **Tembakan:** Bidik kepala atau jantung. Hewan yang terluka akan meninggalkan jejak darah yang harus Anda ikuti untuk menyelesaikan perburuan.
3.  **Menguliti:** Setelah hewan tumbang, berinteraksilah dengan bangkainya untuk mengulitinya. Ini memberikan daging untuk regenerasi kesehatan dan kulit yang dapat dibuat menjadi barang unik, seperti Bear Hat yang didambakan.
4.  **Kesadaran Predator:** Beruang dan serigala jauh lebih cepat daripada pemain. Jika Anda terjatuh dari kuda, gunakan 'Q' untuk dash menjauh dari terkaman dan 'C' untuk melawan dengan tinju jika senjata Anda kosong.

## Progresi Karakter dan Sistem "Prisoner"

Aspek unik dari simulator ini adalah tidak adanya tombol "Reset Character" tradisional. Jika Anda berada dalam posisi yang tidak menguntungkan atau ingin mengubah peran, Anda tidak bisa sekadar respawn.

Ketika seorang pemain mati, mereka mungkin respawn sebagai "Prisoner" (Tahanan) tergantung pada tindakan mereka sebelumnya atau status server saat ini. Tahanan memiliki akses terbatas ke senjata dan harus menemukan cara untuk mendapatkan kebebasan mereka atau menemukan pemain yang bersedia "mengakhiri penderitaan mereka" sehingga mereka dapat respawn sebagai koboi standar. Ini menambah lapisan konsekuensi pada setiap baku tembak, membuat penguasaan **kontrol bridger western** menjadi lebih vital untuk kelangsungan hidup jangka panjang.

| Peran | Akses ke Toko | Perlengkapan Awal | Catatan |
|---|---|---|---|
| **Cowboy** | Akses Penuh | Revolver/Repeater | Peran standar bagi sebagian besar pemain. |
| **Killer** | Terbatas | Loot Tingkat Tinggi | Mendapatkan akses ke kamp bandit tetapi kehilangan hak istimewa di kota. |
| **Prisoner** | Tidak Ada | Tidak Ada | Harus melarikan diri atau dieksekusi untuk mengubah peran. |

## FAQ

**T: Bagaimana cara memperbaiki senjata saya jika berhenti menembak?**

J: Sebagian besar senjata api dalam game mengharuskan Anda untuk mengokang peluru secara manual setelah setiap tembakan atau pengisian ulang. Pastikan Anda telah menekan 'R' untuk mengisi ulang dan kemudian M1 untuk memasukkan peluru ke kamar. Jika senjata masih macet, periksa jumlah amunisi Anda di menu statistik 'M'.

**T: Bisakah saya mengganti pakaian jika saya ditandai sebagai Killer?**

J: Penjahit standar seperti Taylor Scurlock umumnya akan menolak bekerja dengan Killer. Anda perlu menemukan vendor pasar gelap atau menjarah item pakaian dari pemain lain atau bandit AI untuk mengubah penampilan Anda.

**T: Apa cara tercepat untuk bepergian melintasi peta?**

J: Meskipun lari cepat (W+W) berguna untuk jarak pendek, kuda Anda adalah moda transportasi utama. Menggunakan **kontrol bridger western** untuk memanggil (H) dan menunggangi (N) kuda Anda sangatlah penting. Perhatikan bahwa kuda memiliki kesehatan sendiri dan dapat dibunuh oleh predator atau pemain lain.

**T: Apakah game ini membutuhkan Robux untuk dimainkan?**

J: Hingga fase Alpha 2026, game ini biasanya memerlukan biaya masuk kecil sekitar 25 Robux. Harga ini dapat berubah sewaktu-waktu seiring perkembangan game menuju rilis penuh.
