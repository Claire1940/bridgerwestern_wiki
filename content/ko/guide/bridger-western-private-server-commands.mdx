## 개인 서버 시작하기

Bridger Western의 개인 서버는 플레이어가 빌드와 메커니즘을 실험할 수 있는 샌드박스 역할을 합니다. 명령어 콘솔에 접속하려면 일반적으로 서버 소유자이거나 소유자로부터 "관리자(Admin)" 상태를 부여받아야 합니다. 대부분의 명령어는 특정 접두사(일반적으로 콜론(:) 또는 슬래시(/))를 사용하여 게임 내 채팅 인터페이스를 통해 실행됩니다.

### 필수 관리 명령어

개인 서버를 관리하려면 환경을 체계적으로 유지하기 위한 일련의 도구가 필요합니다. 아래 표는 플레이어와 서버 상태를 관리하는 데 사용되는 가장 일반적인 관리용 **Bridger Western 개인 서버 명령어**입니다.

| 명령어 구문 | 수행 작업 | 사용 상황 |
|---|---|---|
| **:kick [플레이어명]** | 서버에서 플레이어를 제거 | 방해되는 유저 추방 |
| **:tp [플레이어1] [플레이어2]** | 플레이어 1을 플레이어 2에게 순간이동 | 그룹 전투 또는 거래 조직 |
| **:bring [플레이어명]** | 플레이어를 내 위치로 순간이동 | 이벤트를 위해 플레이어 소집 |
| **:announce [메시지]** | 모든 플레이어에게 전역 메시지 표시 | 서버 전체 공지 사항 알림 |
| **:time [숫자]** | 게임 내 시간대 변경 | 다양한 조명 아래에서 안력(Ocular Prowess) 테스트 |
| **:kill [플레이어명]** | 대상 플레이어 캐릭터 초기화 | 리스폰 지점 또는 노화 메커니즘 테스트 |

> **⚠️ 경고:** `:kill` 명령어를 과도하게 사용하면 플레이어의 나이가 늘어납니다. Bridger Western에서 노화는 죽을 때마다 발생하며, 나이를 다시 전성기로 줄이려면 독베인(Dog’s Bane) 허브가 필요합니다.

## 개인 서버에서 진행도 최적화하기

플레이어들이 개인 서버에 투자하는 주된 이유 중 하나는 공개 로비에서 흔히 발생하는 "떼거리(swarm)" 효과를 피하기 위해서입니다. 공개 서버에서는 높은 플레이어 밀도 때문에 유해 부위를 통해 스탠드를 얻는 것이 거의 불가능할 수 있습니다. 개인 서버에서도 유해 부위의 스폰 속도는 플레이어 수의 영향을 받지만, 경쟁자가 없기 때문에 원하는 스탠드를 확보하기가 훨씬 쉽습니다.

### 낚시 전략
낚시는 "물라(Moola)"를 벌고 티어(Tier)를 올리는 가장 일관된 방법입니다. 개인 서버에서는 저격당할 걱정 없이 낚시 마을(Fishing Village)이나 타코 미끼 상점(Taco Bait Shop)에서 낚시를 할 수 있습니다.

1.  **장비 구매:** 타코 미끼 상점으로 가서 낚싯대와 미끼를 삽니다.
2.  **물라 파밍:** 잡은 물고기를 NPC에게 다시 팝니다. 희귀한 물고기는 마리당 1,500 물라 이상을 벌어다 줍니다.
3.  **희귀 아이템 찾기:** 낚시를 하면 0.5%의 확률로 스탠드 화살(Stand Arrow)이 드롭되는데, 이는 유해 부위를 사냥하는 것보다 더 안전한(비록 느리지만) 대안입니다.

### 말 관리
개인 서버에서는 시간을 들여 말의 스탯을 "리롤(re-roll)"할 수 있습니다. 기차역 근처 리지 카운티(Ridge County)에 있는 NPC를 방문하세요. 전문가들은 **1,000 물라 이상의 말**만 유지할 것을 권장하는데, 이런 말들이 대개 우수한 속도와 스태미나 스탯을 가지고 있기 때문입니다. 개인 공간을 활용해 Shift를 눌러 질주하고 물속에서 Space를 눌러 강을 빠른 속도로 건너는 "말 기술(Horse Tech)"을 테스트해 보세요.

## 고급 카드 커스터마이징

Bridger Western에는 패시브 버프를 부여하는 "카드" 시스템이 있습니다. 이는 늪지대에 있는 마녀의 오두막(Witch's Hut)을 방문하여 잠금 해제할 수 있습니다. 개인 서버에서는 기습당할 걱정 없이 자유롭게 늪지대로 이동하여 물라를 쓰고 "내 운세 읽기(Read My Fortune)"를 할 수 있습니다.

### 최상위 티어 카드 조합
(특정 관리자 스크립트로 활성화된 경우) **Bridger Western 개인 서버 명령어**를 사용하여 자신에게 물라를 지급하면 최고의 패시브를 위해 카드를 돌릴 수 있습니다.

*   **에임봇 빌드:** *High Noon*, *Evil Eye*, *Folk's High Noon*을 조합하면 안력을 사용하는 동안 조준선이 완벽하게 중앙에 있지 않아도 사격을 명중시킬 수 있습니다.
*   **힐링 빌드:** *Secret Technique*나 강장제(Tonics)의 효과를 강화하는 카드에 집중하세요.
*   **스피드 빌드:** 질주나 승마 시 스태미나 소모를 줄여주는 패시브를 찾으세요.

> **💡 팁:** 티어 3에서는 최대 3개의 카드를 활성화할 수 있습니다. 마음에 들지 않는 카드가 나오면 단순히 "이 운명을 거부하기(Reject this Fate)"를 선택하고 다시 돌리면 됩니다.

## 자주 묻는 질문 (FAQ)

**Q: 소유자가 아닌데 어떻게 Bridger Western 개인 서버 명령어를 활성화하나요?**

A: 서버 소유자가 수동으로 관리자 권한을 부여해야 합니다. 이는 대개 `:admin [사용자명]`과 같은 소유자 전용 명령어를 통해 이루어집니다. 권한을 받으면 표준 명령어 접두사를 사용하여 작업을 실행할 수 있습니다.

**Q: 개인 서버에서 스탠드를 더 빨리 얻을 수 있나요?**

A: 네. 유해 부위의 스폰율은 기술적으로 플레이어가 많은 서버에서 더 높지만(플레이어가 많을수록 30초마다 스폰), 개인 서버에서는 경쟁이 없기 때문에 부위가 나타나기만 하면 확실하게 얻을 수 있습니다. 또한 완전히 안전하게 0.5% 확률의 스탠드 화살 낚시를 할 수도 있습니다.

**Q: 개인 서버와 공개 서버 간에 스탯과 물라가 저장되나요?**

A: 네, Bridger Western은 글로벌 저장 시스템을 사용합니다. **Bridger Western 개인 서버 명령어**나 수동 파밍을 통해 얻은 레벨, 돈, 말, 스탠드를 포함한 모든 진행 상황은 어떤 서버에 접속하든 그대로 유지됩니다.

**Q: 티어를 올리는 가장 빠른 방법은 무엇인가요?**

A: 가장 빠른 레벨업 방법은 낚시와 월드 이벤트 참여를 병행하는 것입니다. 개인 서버에서는 월드 이벤트를 직접 트리거하거나 타코 미끼 상점에서 단순히 물고기 파밍을 하여 몇 시간의 집중적인 플레이만으로 티어 3(최고 레벨)에 도달할 수 있습니다.
