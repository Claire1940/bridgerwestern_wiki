## 유해 부위 방법: 고속 획득

스탠드를 획득하는 가장 직접적이고 빠른 방법은 유해 부위 시스템을 이용하는 것입니다. 이 방법은 위험이 따르는 PvP를 즐기고 "전면전"을 두려워하지 않는 플레이어를 위해 설계되었습니다. 유해 부위가 스폰될 때는 조용히 나타나지 않습니다. 게임이 모든 플레이어에게 이를 알리며, 해당 지역을 순식간에 전투 지역으로 만듭니다.

### 스폰 메커니즘 이해하기
유해 부위는 특정 서버 조건에 따라 관리됩니다. 플레이어의 상호작용 없이 무작위 간격으로 단순히 나타나는 것이 아닙니다. 효율성을 극대화하려면 "최소 플레이어 규칙"을 이해해야 합니다.

| 특징 | 요구 사항 / 세부 사항 |
| :--- | :--- |
| **최소 플레이어 수** | 서버에 최소 **3명의 플레이어**가 있어야 함 |
| **스폰 빈도** | 약 **30초**마다 |
| **시각적 표시** | 하늘에서 내려오는 거대한 **황금색 빛의 줄기** |
| **서버 알림** | 모든 플레이어에게 전역 공지 텍스트 표시 |
| **성공률** | 융합 과정에서 살아남을 수만 있다면 매우 높음 |

### 융합 과정
황금색 빛의 줄기를 찾아 유해 부위를 습득하면 진짜 도전이 시작됩니다. 즉시 스탠드를 얻는 것이 아니라, 해당 부위와 "융합"해야 합니다. 이 시간 동안 여러분은 맵의 모든 플레이어에게 주요 표적이 됩니다.

1.  **습득:** 부위에 접근하여 상호작용합니다.
2.  **추격:** 여러분의 위치가 사실상 노출되며, 플레이어들이 여러분을 죽이기 위해 달려들 것입니다.
3.  **위험 요소:** 부위를 소지한 상태에서 대미지를 입으면 부위를 떨어뜨릴 확률이 매우 높으며, 다른 사람이 이를 훔칠 수 있습니다.
4.  **완료:** 융합 바가 채워지면 팝업이 나타나고 인터페이스에 새로운 "스탠드" 단축키가 추가됩니다.

## 낚시 방법: 인내의 길

유해 부위 사냥의 혼란이 적성에 맞지 않는다면, 낚시는 훨씬 느리지만 평화로운 대안을 제공합니다. 낚시를 통해 플레이어는 스탠드 화살(Stand Arrows)이나 오토 샤드(Auto Shards)와 같은 전설적인 아이템을 낚을 수 있습니다. 이는 솔로 플레이어나 끊임없는 PvP를 피하고 싶은 분들이 선호하는 방법입니다.

### 타코 미끼 상점(Taco Bait Shop) 위치 찾기
낚시를 시작하기 전에 적절한 장비가 필요합니다. 타코 미끼 상점은 브리저 웨스턴의 모든 해상 관련 활동의 중심지입니다. 맵이 처음이라면 기차 철로를 따라가는 것이 가장 쉬운 방법입니다.

*   **말 이용:** **H** 키를 눌러 말을 소환하고 상점역에 도착할 때까지 철로를 따라갑니다.
*   **기차 이용:** 아무 역에서나 기다렸다가 타코 베이(Taco Bay) 정류장으로 가는 기차를 직접 탑승합니다.

### 낚시 장비 및 비용
상점 안의 NPC와 대화하여 기본 장비를 구매해야 합니다. 맨손으로는 전설 아이템을 잡을 수 없습니다.

| 아이템 | 용도 | 비용/희귀도 |
| :--- | :--- | :--- |
| **낚싯대** | 낚시에 필요한 기본 도구 | 표준 화폐 |
| **미끼** | 획득률 증가/입질 유도 | 소모품 |
| **스탠드 화살** | 무작위 스탠드 부여 | **0.5% 드랍률** |
| **오토 샤드** | 스탠드용 전설급 획득 아이템 | **0.5% 드랍률** |

### 성공적으로 낚시하는 법
브리저 웨스턴의 낚시는 타이밍 미니게임입니다. 낚싯대를 장착하고 미끼를 부착했다면 아무 물가나 찾으세요.
1.  마우스 왼쪽 클릭으로 낚싯줄을 던집니다.
2.  찌 근처에 파란색 표시나 물방울이 튀는 것을 지켜보세요.
3.  표시가 움직이거나 입질이 확인되면 즉시 **마우스 왼쪽 클릭**을 하여 낚아 올립니다.
4.  운 좋게 화살이나 샤드를 낚았다면 즉시 사용하세요.

> **팁:** 스탠드 화살과 샤드는 은행에 보관하거나 저장할 수 없습니다. 인벤토리에 있는 상태에서 서버를 나가거나 죽으면 영원히 사라집니다. 새 화살을 사용하기 전에 현재 장착된 스탠드가 없는지 확인하세요.

## 스탠드 헌터를 위한 프로 전략

브리저 웨스턴의 세계를 탐험하려면 어디로 가야 하는지 아는 것 이상으로, 게임 환경을 자신에게 유리하게 조작하는 법을 알아야 합니다.

### 효율성을 위한 서버 이동(Server Hopping)
15명 이상의 플레이어가 단 하나의 유해 부위를 두고 싸우는 "메가 서버"에 있다면 성공 확률이 크게 떨어집니다. **브리저 웨스턴에서 스탠드를 얻는 방법**을 최적화하려면 서버 브라우저를 사용하세요.
1.  **서버 탭**을 엽니다.
2.  **오름차순/내림차순(Ascend/Descend)** 토글을 클릭하여 플레이어 수로 정렬합니다.
3.  플레이어가 **정확히 3명 또는 4명**인 서버에 접속합니다.
4.  이렇게 하면 유해 부위 스폰을 위한 최소 요구 사항을 충족하면서도 싸워야 할 적의 수를 최소화할 수 있습니다.

### 스탠드 관리하기
성공적으로 스탠드를 획득했다면 사용법을 알아야 합니다. 스탠드는 단순한 패시브 버프가 아니라 활동적인 전투원입니다.

*   **소환:** 지정된 단축키(보유한 경우 보통 'Q' 또는 특정 UI 버튼)를 사용하여 스탠드를 소환합니다.
*   **능력:** 대부분의 스탠드는 E, R, T, F에 매핑된 3~4개의 고유 기술을 가지고 있습니다.
*   **해제:** 스탠드 화살을 통해 다른 스탠드를 시도하려면 먼저 특정 NPC와 대화하거나 "스탠드 초기화(Stand Reset)" 아이템을 사용해야 합니다. 두 개의 스탠드를 동시에 보유할 수 없기 때문입니다.

## FAQ

**Q: 스탠드 화살을 친구에게 줄 수 있나요?**

A: 아니요. 낚시를 통해 얻은 스탠드 화살과 오토 샤드는 떨어뜨릴 수 없으며 거래도 불가능합니다. 또한 사망 시 소실되므로 낚는 즉시 사용하세요.

**Q: 왜 제 서버에서는 유해 부위가 스폰되지 않나요?**

A: 가장 흔한 이유는 플레이어 수입니다. 30초 타이머가 작동하려면 서버에 최소 3명의 활성 플레이어가 있어야 합니다. 누군가 나가서 인원이 2명으로 줄어들면 스폰이 중단됩니다.

**Q: 신규 플레이어에게 브리저 웨스턴에서 스탠드를 얻는 가장 빠른 방법은 무엇인가요?**

A: 가장 빠른 방법은 인원이 적은 서버(3~4명)에 들어가 유해 부위를 사냥하는 것입니다. 낚시는 더 안전하지만 0.5%의 드랍률 때문에 몇 시간이 걸릴 수 있는 반면, 유해 부위는 30초마다 스폰됩니다.

**Q: 죽으면 스탠드를 잃게 되나요?**

A: 아니요. 융합에 성공하거나 화살을 통해 획득한 스탠드는 여러분이 직접 초기화하거나 진화시키지 않는 한 캐릭터에 영구적으로 귀속됩니다.
