<Section icon={Sparkles} title="Configurando tu Macro de Auto Pesca de Bridger Western">
Para configurar con éxito un macro de **auto pesca** para Bridger Western, son necesarios ciertos requisitos previos y configuraciones. El proceso a menudo implica descargar software de terceros y ajustar la configuración del juego para garantizar un rendimiento y una detección óptimos.

<Steps>
  <Step num="1" title="Descarga el Software del Macro">
    Accede al servidor oficial de Discord (enlazado en las guías relevantes) y localiza la sección de macros gratuitas. Busca "Bridger" y descarga el archivo EXE de instalación desde el enlace de GitHub proporcionado. Ten cuidado con las advertencias del antivirus y elige "Ejecutar de todos modos" si se te solicita, o desactiva temporalmente tu antivirus y añade una exclusión.
  </Step>
  <Step num="2" title="Instala e Inicia el Macro">
    Haz doble clic en el EXE descargado para ejecutar el instalador. Sigue las indicaciones en pantalla, haciendo clic en "Siguiente" hasta que se complete. Una vez instalado, inicia la aplicación del macro. Asegúrate de que esté actualizada a la última versión si se te solicita.
  </Step>
  <Step num="3" title="Configura los Ajustes Básicos">
    Ajusta la superposición de la región F1 para que coincida con la pantalla de tu minijuego. Establece F3 como la tecla de inicio/parada del macro. Selecciona tu monitor principal. Habilita "Siempre encima" para mantener la ventana del macro visible. Configura tu dispositivo de salida para que coincida con la configuración de sonido de tu sistema y maximiza el volumen del juego para una mejor detección.
  </Step>
  <Step num="4" title="Ajusta la Detección y los Retrasos">
    Para el método de detección, generalmente se recomienda NumPy para la estabilidad; si falla, prueba OpenCV. Ajusta la "Tolerancia de píxeles" para el reconocimiento de color. Establece el "Retraso de escaneo" (por ejemplo, 0.05 segundos) y los "Escaneos en blanco para salir del minijuego" (por ejemplo, 5 veces). Considera habilitar "Omitir minijuego" para fallar a propósito y volver a lanzar si lo prefieres.
  </Step>
  <Step num="5" title="Posición de Lanzamiento y Cebo Personalizados">
    Utiliza la función "Posición de lanzamiento personalizada" para definir dónde lanza tu personaje. Asegúrate de que tu caña de pescar esté en la primera ranura del inventario y el cebo en la segunda, o ajusta la configuración dentro del macro según corresponda.
  </Step>
</Steps>

<Callout type="warning" title="Interferencia del Antivirus">
Si tu antivirus bloquea el macro, deberás deshabilitarlo temporalmente o crear una exclusión de carpeta. Consulta la documentación de tu software antivirus para obtener instrucciones específicas sobre cómo hacerlo.

| Problema | Causa Potencial | Solución |
|---|---|---|
| **El macro no se inicia** | Atajos de teclado incorrectos, conflictos de software o versión desactualizada. | Asegúrate de que F3 esté configurado como inicio/parada, cierra las aplicaciones en conflicto, actualiza el macro. |
| **El minijuego de pesca no se completa** | Configuración de detección inexacta o cambios en la interfaz de usuario. | Ajusta la "Tolerancia de píxeles", prueba la detección "NumPy" u "OpenCV", cambia el "Modo de temporización" a "Adaptativo" si está disponible. |
| **El antivirus bloquea el macro** | Detección de falso positivo. | Deshabilita el antivirus o añade una exclusión para la carpeta del macro. |
| **El macro se detiene aleatoriamente** | Anti-AFK no habilitado, retraso del servidor o script desactualizado. | Habilita Anti-AFK, verifica la conexión a internet, asegúrate de que el script sea la última versión. |
| **La posición de lanzamiento personalizada no funciona** | Coordenadas incorrectas o desplazamiento de la interfaz de usuario del juego. | Vuelve a configurar la posición de lanzamiento personalizada con cuidado, asegúrate de que la escala de resolución sea del 100%. |
| **Estado "Desconectado" o "Conectando"** | Problemas del servidor backend para el macro. | Termina el proceso del macro, reinicia e intenta reconectar. Reinstala si persiste. |
